[Tue 2026-02-24 00:06 GMT+8] OpenRouter 如何获取免费的Claude额度
assistant: OpenRouter 获取免费额度步骤：

---

## 1. 注册账号

访问 https://openrouter.ai/
- 邮箱注册（支持国内邮箱）
- 或使用 Google/GitHub 账号

---

## 2. 获取免费额度

**新用户：**
- 注册后通常有 **$5-10 信用额度**
- 有效期：通常是 30 天

**充值奖励（可选）：**
- 首次充值可能有额外赠送
- 但这不是必须的，免费额度先用完再说

---

## 3. 创建 API Key

1. 登录后进入 **Settings** → **API Keys**
2. 点击 **Create API Key**
3. 复制保存（格式：`sk-or-v1-...`）

---

## 4. 配置使用 Claude

OpenRouter 支持 Claude 3.5/3.7，调用方式：

```bash
# 设置环境变量
$env:OPENROUTER_API_KEY="sk-or-v1-你的密钥"

# 测试调用
curl https://openrouter.ai/api/v1/chat/completions \
  -H "Authorization: Bearer $env:OPENROUTER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "anthropic/claude-3.5-sonnet",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```

---

## 5. 与 OpenClaw 集成

可以在 `~/.openclaw/openclaw.json` 添加 OpenRouter 作为 provider：

```json5
{
  models: {
    providers: {
      openrouter: {
        baseUrl: "https://openrouter.ai/api/v1",
        apiKey: "${OPENROUTER_API_KEY}",
        models: [
          { id: "anthropic/claude-3.5-sonnet", alias: "Claude 3.5" }
        ]
      }
    }
  }
}
```
