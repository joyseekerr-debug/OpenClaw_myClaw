# 48. 期权基础 (Options Basics)

## 概念定义

期权（Option）是一种金融衍生工具，赋予持有者在特定日期或之前以特定价格买入或卖出标的资产的权利（但非义务）。

## 期权基本要素

### 核心概念
```
1. 买方（权利方）
   - 支付权利金（Premium）
   - 获得权利，无义务
   - 最大损失 = 权利金

2. 卖方（义务方）
   - 收取权利金
   - 承担履约义务
   - 风险可能很大

3. 标的资产
   - 股票、指数、商品等
   - 期权价值的基础

4. 行权价（Strike Price）
   - 约定的买卖价格
   - 期权合约的核心参数

5. 到期日（Expiration Date）
   - 期权有效的最后日期
   - 过期后期权失效

6. 权利金（Premium）
   - 期权的价格
   - 买方支付给卖方的费用
```

## 期权类型

### 按权利分类
```
看涨期权（Call Option）：
- 买方权利：以行权价买入标的
- 适用场景：预期上涨
- 买方收益：理论上无限
- 买方风险：权利金

看跌期权（Put Option）：
- 买方权利：以行权价卖出标的
- 适用场景：预期下跌
- 买方收益：行权价-0（最大）
- 买方风险：权利金
```

### 按行权时间分类
```
欧式期权：
- 只能在到期日行权
- 结构简单
- A股ETF期权多为欧式

美式期权：
- 到期前任何时间可行权
- 灵活性更高
- 通常价格略高
```

## 期权状态

### 内在价值状态
```
看涨期权：
├── 实值（ITM）：标的价 > 行权价
│   └── 内在价值 = 标的价 - 行权价
├── 平值（ATM）：标的价 ≈ 行权价
│   └── 内在价值 = 0
└── 虚值（OTM）：标的价 < 行权价
    └── 内在价值 = 0

看跌期权：
├── 实值（ITM）：标的价 < 行权价
│   └── 内在价值 = 行权价 - 标的价
├── 平值（ATM）：标的价 ≈ 行权价
└── 虚值（OTM）：标的价 > 行权价
```

## 期权价格构成

### 权利金 = 内在价值 + 时间价值
```
内在价值：
- 立即行权可获得的收益
- 实值期权才有
- ≥ 0

时间价值：
- 到期前价格波动带来的价值
- 所有期权都有（到期前）
- 受时间、波动率影响
- 到期时 = 0

示例：
股票现价：105元
看涨期权行权价：100元
权利金：8元

内在价值 = 105 - 100 = 5元
时间价值 = 8 - 5 = 3元
```

## 期权 vs 股票

| 维度 | 股票 | 期权 |
|------|------|------|
| 权利义务 | 所有权 | 选择权（买方）/义务（卖方） |
| 资金需求 | 全额 | 权利金（杠杆） |
| 风险收益 | 线性 | 非线性 |
| 期限 | 永久 | 有到期日 |
| 收益潜力 | 上涨无限 | 买方无限/卖方有限 |
| 亏损可能 | 归零 | 买方有限/卖方很大 |

## 期权基本策略

### 买入看涨（Long Call）
```
适用：强烈看涨
成本：支付权利金
收益：理论上无限
风险：权利金全部损失

盈亏平衡点：行权价 + 权利金
```

### 买入看跌（Long Put）
```
适用：强烈看跌
成本：支付权利金
收益：行权价 - 0 - 权利金
风险：权利金全部损失

盈亏平衡点：行权价 - 权利金
```

### 卖出看涨（Short Call）
```
适用：看跌或横盘
收入：收取权利金
收益：权利金（最大）
风险：理论上无限

注意：风险极大，需保证金
```

### 卖出看跌（Short Put）
```
适用：看涨或横盘
收入：收取权利金
收益：权利金（最大）
风险：行权价 - 0 - 权利金

用途：低价建仓、增强收益
```
