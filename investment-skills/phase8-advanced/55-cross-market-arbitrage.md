# 55. 跨市场套利 (Cross-Market Arbitrage)

## 概念定义

跨市场套利是指利用同一资产在不同市场之间的价格差异，同时进行买卖操作以获取无风险或低风险收益的交易策略。

## 套利类型

### 1. 同一资产跨市场套利

#### A股-港股套利
```
标的：同时在A股和港股上市的公司（如招商银行、中国平安）

套利逻辑：
- A股价格 vs 港股价格（换算后）
- 价差过大时存在套利机会

障碍：
- 无法直接转换股份
- 需要两个市场都有持仓
- 汇率风险
- 资金门槛

近似套利：
- 通过ETF间接套利
- 例如：AH股溢价ETF
```

#### ADR套利
```
标的：美国存托凭证（ADR）vs 本土股票

示例：
- 阿里巴巴ADR（NYSE）
- 阿里巴巴港股（HKEX）

套利方式：
- ADR价格与港股价格换算后对比
- 价差大时可套利

优势：
- 部分ADR可转换
- 流动性好
```

### 2. 相关资产跨市场套利

#### ETF-一揽子股票套利
```
原理：
- ETF价格 vs 成分股市值
- 价差大时存在套利空间

正向套利（ETF溢价）：
- 买入一揽子成分股
- 申购ETF份额
- 卖出ETF

反向套利（ETF折价）：
- 买入ETF
- 赎回成分股
- 卖出股票

门槛：
- 资金量大
- 需要程序化交易
- 交易速度快
```

#### 期货-现货套利
```
原理：
- 期货价格 vs 现货价格
- 理论上期货价格 = 现货价格 + 持有成本

正向套利（期货溢价）：
- 买入现货
- 卖出期货
- 到期交割

反向套利（期货折价）：
- 卖出现货
- 买入期货
- 到期交割

注意：
- 持有成本计算
- 到期时间匹配
- 保证金管理
```

### 3. 跨品种套利

#### 相关商品套利
```
示例：黄金相关
- 黄金期货（COMEX）
- 黄金ETF（GLD）
- 黄金矿业股
- 金矿ETF（GDX）

逻辑：
- 正常情况下相关性高
- 短期偏离提供套利机会

风险：
- 相关性可能破裂
- 基本面变化
- 非纯套利
```

## 跨市场套利实施

### 套利识别
```
监控指标：
- 价差历史分布
- 价差均值回归特性
- 当前价差位置

触发条件：
- 价差超过历史均值2个标准差
- 价差开始回归
- 交易成本覆盖
```

### 风险控制
```
1. 执行风险
   - 两笔交易无法同时成交
   - 价差在下单后扩大
   - 解决方案：程序化交易、限价单

2. 资金风险
   - 需要双边保证金
   - 汇率风险（跨国）
   - 解决方案：充足资金、对冲汇率

3. 市场风险
   - 相关性破裂
   - 极端行情
   - 解决方案：严格止损、分散套利
```

### 套利成本
```
显性成本：
- 交易佣金
- 印花税
- 过户费
- 申购赎回费

隐性成本：
- 买卖价差
- 滑点
- 资金成本
- 机会成本

净套利收益 = 毛价差 - 总成本
```

## 跨市场套利案例

### 案例1：港股-ADR套利
```
标的：阿里巴巴

情景：
- 港股：100港元
- ADR：103港元（换算后）
- 价差：3%

操作：
- 买入港股阿里巴巴
- 卖出ADR阿里巴巴
- 持有至价差回归

风险：
- 汇率波动
- 价差可能扩大
- 资金占用成本
```

### 案例2：ETF套利
```
标的：沪深300ETF

情景：
- ETF价格：4.00元（溢价1%）
- 成分股市值：3.96元

操作：
- 买入成分股组合
- 申购ETF份额
- 卖出ETF

收益：
- 毛利润：1%
- 成本：0.3%
- 净利润：0.7%
```

## 跨市场套利注意事项

### 1. 不是无风险
```
风险来源：
- 执行风险
- 模型风险
- 极端行情
- 监管变化

应有风险意识，严格风控
```

### 2. 需要专业设备
```
要求：
- 低延迟系统
- 多市场接入
- 程序化交易
- 高速网络

门槛：
- 技术投入大
- 机构为主
```

### 3. 机会稀少
```
现状：
- 市场竞争激烈
- 套利空间缩小
- 需要大量资金
- 高频低利润

适合：
- 专业机构
- 量化基金
- 券商自营
```
