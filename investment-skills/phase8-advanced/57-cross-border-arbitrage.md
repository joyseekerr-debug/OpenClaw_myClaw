# 57. 跨境套利 (Cross-Border Arbitrage)

## 概念定义

跨境套利是指利用同一资产在不同国家/地区市场之间的价格差异，通过跨市场交易获取收益的策略。由于汇率、监管、税收等因素，同一资产可能在不同市场存在价差。

## 跨境套利类型

### 1. 股票跨境套利

#### A股-H股套利
```
标的：同时在A股和港股上市的公司（AH股）

价差来源：
- 市场分割
- 投资者结构差异
- 流动性差异
- 汇率预期

套利方式：
直接套利（受限）：
- 无法直接转换股份
- 需要两边都有持仓

间接套利：
- 通过AH股溢价指数ETF
- 做多折价股，做空溢价股
- 统计套利

风险：
- 价差可能长期存在
- 汇率风险
- 相关性破裂
```

#### 中概股回归套利
```
背景：
- 美股上市中概股回归港股/A股
- 预期两地上市溢价

套利逻辑：
- 持有美股
- 等待回归港股
- 港股通常溢价

风险：
- 回归时间不确定
- 回归可能失败
- 政策风险
```

### 2. ETF跨境套利

#### 同一ETF多市场上市
```
示例：
- 南方东英恒指ETF（港股）
- 相同ETF在新加坡、伦敦上市

套利条件：
- 不同时区价格差异
- 汇率换算后价差
- 交易时间重叠少

实施难度：
- 需要多市场账户
- 资金跨境
- 汇率对冲
```

#### 跨境ETF折溢价套利
```
境内上市跨境ETF：
- 中概互联网ETF
- 纳指ETF
- 恒指ETF

套利方式：
场内价格 vs IOPV（参考净值）

溢价套利：
- 申购ETF（用一揽子股票）
- 场内卖出

折价套利：
- 场内买入ETF
- 赎回股票卖出

限制：
- 外汇额度
- 申赎门槛
- T+2交割
```

### 3. 外汇套利

#### 三角套利
```
原理：
- 三种货币对交叉汇率不一致
- 通过循环兑换获利

示例：
USD/CNY = 7.0
EUR/USD = 1.1
EUR/CNY = 7.8

理论EUR/CNY = 7.0 × 1.1 = 7.7
实际7.8 > 7.7

套利：
1. 用人民币买美元
2. 用美元买欧元
3. 用欧元买人民币
4. 获利 = 7.8/7.7 - 1 = 1.3%

现实：
- 市场效率高效
- 机会极少
- 需要高频系统
```

#### 利率平价套利
```
原理：
- 考虑利率差异的远期汇率
- 实际远期与理论远期偏离

 Covered Interest Rate Parity:
 F = S × (1 + r_d) / (1 + r_f)

套利：
- 当实际远期偏离理论值
- 借入低息货币
- 投资高息货币
- 远期锁定汇率
```

## 跨境套利实施要点

### 账户与资金
```
要求：
- 多市场交易账户
- 跨境资金通道
- 外汇额度

挑战：
- 开户复杂
- 资金出入境限制
- 税务申报
```

### 汇率风险管理
```
风险：
- 套利期间汇率波动
- 可能抵消套利收益

对冲：
- 远期外汇合约
- 外汇期货
- 自然对冲（两边都有资金）
```

### 税务考虑
```
涉及税种：
- 预扣税（股息）
- 资本利得税
- 印花税
- 增值税

筹划：
- 利用税收协定
- 选择税收友好地区
- 税务合规
```

## 跨境套利案例

### 案例1：黄金跨境套利
```
市场：
- 伦敦金（美元计价）
- 上海金（人民币计价）

套利：
- 换算后价格对比
- 考虑汇率预期
- 实物进出口成本

实施：
- 进口/出口黄金
- 涉及海关、运输、保险
- 适合大型机构
```

### 案例2：比特币跨境套利
```
市场：
- 不同交易所价格差异
- 不同国家价格差异

套利：
- 低价交易所买入
- 高价交易所卖出
- 转移数字资产

风险：
- 转移时间
- 交易所风险
- 监管风险
```

## 跨境套利注意事项

### 监管合规
```
重要：
- 遵守各市场监管
- 外汇管理规定
- 反洗钱合规
- 信息披露要求

违规风险：
- 账户冻结
- 罚款
- 法律责任
```

### 政治风险
```
风险：
- 贸易政策变化
- 资本管制加强
- 制裁风险

应对：
- 分散国家
- 关注政策
- 应急预案
```

### 机会评估
```
评估因素：
- 价差大小
- 持续性
- 实施成本
- 风险水平

结论：
- 散户难以参与
- 机构专属机会
- 需要专业团队
- 系统化操作
```
