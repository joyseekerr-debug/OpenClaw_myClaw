# 51. 期权希腊字母 (Option Greeks)

## 希腊字母概述

希腊字母用于衡量期权价格对各因素的敏感度，是期权风险管理和头寸管理的核心工具。

```
主要希腊字母：
├── Delta (Δ) - 标的资产价格敏感度
├── Gamma (Γ) - Delta的变化率
├── Theta (Θ) - 时间衰减
├── Vega (V) - 波动率敏感度
└── Rho (ρ) - 利率敏感度
```

## Delta (Δ)

### 定义
期权价格对标的资产价格的一阶导数。

### 特性
```
看涨期权：0 ≤ Δ ≤ 1
看跌期权：-1 ≤ Δ ≤ 0

近似含义：
- 标的涨1元，期权涨Δ元
- 可视为期权到期实值的概率

状态与Delta：
┌─────────┬──────────┬──────────┐
│  状态   │ 看涨Delta │ 看跌Delta │
├─────────┼──────────┼──────────┤
│ 深度实值 │   ≈1     │   ≈-1    │
│ 平值    │   ≈0.5   │   ≈-0.5  │
│ 深度虚值 │   ≈0     │   ≈0     │
└─────────┴──────────┴──────────┘
```

### 应用
```
对冲（Delta Hedging）：
- 持有看涨期权，Delta=0.5
- 卖空0.5股股票对冲
- 组合Delta=0，中性

方向敞口：
- Delta = 实际股票敞口
- 看涨正Delta → 看涨
- 看跌负Delta → 看跌
```

## Gamma (Γ)

### 定义
Delta对标的资产价格的变化率，二阶导数。

### 特性
```
买方Gamma > 0
卖方Gamma < 0

平值期权Gamma最大
深度实值/虚值Gamma趋近于0

到期越近，Gamma越大
```

### Gamma风险
```
Gamma Scalping：
- 高Gamma头寸
- 价格大幅波动有利
- 适合预期大波动

Gamma风险：
- 卖出期权（负Gamma）
- 价格大幅波动损失大
- 需要频繁调整对冲
```

## Theta (Θ)

### 定义
期权价格对时间的变化率，时间衰减。

### 特性
```
买方Theta < 0（时间损耗）
卖方Theta > 0（时间收益）

平值期权Theta绝对值最大
到期越近，衰减越快

Theta与Gamma通常反向
- 高Gamma → 高负Theta
- 时间损耗换价格敏感度
```

### 时间衰减规律
```
距到期   衰减速度
────────────────────
3个月    较慢
1个月    加快
1周      很快
最后1天  极速

结论：
- 买方避免持有到期
- 卖方喜欢时间流逝
```

## Vega (V)

### 定义
期权价格对波动率的变化率。

### 特性
```
买方Vega > 0
卖方Vega < 0

平值期权Vega最大
到期越远，Vega越大

Vega通常以1%波动率变化计算
```

### 波动率交易
```
做多波动率：
- 买入跨式组合
- Vega为正
- 波动率上升获利

做空波动率：
- 卖出跨式组合
- Vega为负
- 波动率下降获利
```

## Rho (ρ)

### 定义
期权价格对利率的变化率。

### 特性
```
看涨Rho > 0（利率↑ → 期权↑）
看跌Rho < 0（利率↑ → 期权↓）

影响相对较小（短期）
长期期权影响更大
```

## 希腊字母汇总表

| 希腊字母 | 含义 | 买方符号 | 卖方符号 | 最大值位置 | 风险 |
|----------|------|----------|----------|------------|------|
| Delta | 价格敏感度 | + | - | 实值 | 方向风险 |
| Gamma | Delta变化率 | + | - | 平值、近月 | 大幅波动风险 |
| Theta | 时间衰减 | - | + | 平值、近月 | 时间损耗 |
| Vega | 波动率敏感度 | + | - | 平值、远月 | 波动风险 |
| Rho | 利率敏感度 | ± | ∓ | 实值、远月 | 利率风险 |

## 希腊字母计算示例

```
假设：
股票现价：100元
看涨期权行权价：100元
权利金：5元
到期：1个月

希腊字母估计值：
Delta: +0.52
Gamma: +0.05
Theta: -0.08（每天衰减0.08元）
Vega: +0.12（波动率每升1%，期权涨0.12元）

情景分析：
1. 股价涨到101元
   期权价值 ≈ 5 + 0.52 = 5.52元

2. 持有1天，股价不变
   期权价值 ≈ 5 - 0.08 = 4.92元

3. 波动率从20%升到21%
   期权价值 ≈ 5 + 0.12 = 5.12元
```
