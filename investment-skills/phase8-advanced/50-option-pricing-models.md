# 50. 期权定价模型 (Option Pricing Models)

## 期权定价基础

### 影响期权价格的因素
```
1. 标的资产价格（S）
   - 看涨：S↑ → 期权价格↑
   - 看跌：S↑ → 期权价格↓

2. 行权价（K）
   - 看涨：K↑ → 期权价格↓
   - 看跌：K↑ → 期权价格↑

3. 到期时间（T）
   - 时间↑ → 期权价格↑
   - 时间价值

4. 波动率（σ）
   - 波动率↑ → 期权价格↑
   - 对买卖双方机会都增加

5. 无风险利率（r）
   - 利率↑ → 看涨期权↑
   - 利率↑ → 看跌期权↓

6. 股息（q）
   - 股息↑ → 看涨期权↓
   - 股息↑ → 看跌期权↑
```

## Black-Scholes模型

### 模型简介
```
提出者：Fischer Black, Myron Scholes (1973)
贡献：Robert Merton
荣誉：1997年诺贝尔经济学奖

适用范围：
- 欧式期权
- 无股息（可调整）
- 假设波动率恒定
- 假设价格连续变动（无跳空）
```

### 公式
```
看涨期权：
C = S × N(d1) - K × e^(-rT) × N(d2)

看跌期权：
P = K × e^(-rT) × N(-d2) - S × N(-d1)

其中：
d1 = [ln(S/K) + (r + σ²/2)T] / (σ√T)
d2 = d1 - σ√T

变量：
S = 标的资产现价
K = 行权价
T = 到期时间（年）
r = 无风险利率
σ = 波动率
N() = 标准正态分布累积函数
```

### 计算示例
```
参数：
S = 100（股票现价）
K = 100（行权价）
T = 0.25（3个月）
r = 5%（年化）
σ = 20%（年化波动率）

计算：
d1 = [ln(100/100) + (0.05 + 0.2²/2)×0.25] / (0.2×√0.25)
   = [0 + 0.025] / 0.1
   = 0.25

d2 = 0.25 - 0.2×0.5 = 0.15

N(d1) = N(0.25) ≈ 0.5987
N(d2) = N(0.15) ≈ 0.5596

看涨期权价格：
C = 100×0.5987 - 100×e^(-0.05×0.25)×0.5596
  = 59.87 - 100×0.9876×0.5596
  = 59.87 - 55.27
  = 4.60元
```

## 二叉树模型

### 模型简介
```
提出者：Cox, Ross, Rubinstein (1979)

特点：
- 直观易懂
- 适用美式期权
- 可处理复杂条件
- 计算精度与步数相关
```

### 单期二叉树
```
假设：
- 当前股价：S
- 上涨后：S×u (u > 1)
- 下跌后：S×d (d < 1)
- 上涨概率：p
- 下跌概率：1-p

风险中性概率：
p = (e^(rΔt) - d) / (u - d)

期权价格：
C = e^(-rΔt) × [p×Cu + (1-p)×Cd]

其中Cu、Cd为上涨、下跌后的期权价值
```

### 多期二叉树
```
        S
       / \
     Su   Sd
     /\   /\
   Su² Sud Sd²
   ...

从末端倒推计算每个节点的期权价值
```

## 隐含波动率（Implied Volatility）

### 概念
```
定义：
- 从期权市场价格反推的波动率
- 反映市场对未来的波动预期

计算：
- 已知期权价格、S、K、T、r
- 通过BS公式反解σ
- 需数值方法（如牛顿迭代）
```

### 隐含波动率的应用
```
1. 期权估值
   - 对比隐含波动率与历史波动率
   - 判断期权贵还是便宜

2. 市场情绪
   - 隐含波动率↑ → 预期波动加大
   - 隐含波动率↓ → 预期平静

3. 波动率交易
   - 买卖波动率（非方向）
   - Vega敞口管理
```

### 波动率微笑
```
现象：
- 同一到期日、不同行权价的期权
- 隐含波动率不同
- 图形呈"微笑"或"偏斜"

原因：
- 极端价格变动概率高于正态分布
- 市场恐慌/贪婪
- 供需不平衡

意义：
- BS模型的局限性
- 市场真实风险分布
- 定价需调整
```
