# 52. 期权策略 - 保护性看跌与备兑看涨 (Protective Put & Covered Call)

## 保护性看跌（Protective Put）

### 策略构成
```
组合：持有股票 + 买入看跌期权

成本：股票成本 + 权利金

效果：为股票下跌购买保险
```

### 盈亏分析
```
最大收益：理论上无限（股票上涨）
最大损失：股票成本 - 行权价 + 权利金

盈亏平衡点：股票成本 + 权利金

图形：
利润
 │╲
 │ ╲_______
 │  │      ╲_____
-│__│______________ 股票价格
   行权价  成本+权利金

特点：
- 下跌保护
- 上涨参与（减权利金成本）
- 类似保险
```

### 应用示例
```
情景：
- 持有100股某股票，成本价100元
- 担心短期下跌
- 买入1张看跌期权（行权价95元，权利金2元）

结果分析：
1. 股价跌到80元
   - 股票亏损：(80-100)×100 = -2000元
   - 期权盈利：(95-80-2)×100 = +1300元
   - 净亏损：700元（而非2000元）

2. 股价涨到120元
   - 股票盈利：(120-100)×100 = +2000元
   - 期权价值：0（虚值）
   - 净盈利：2000 - 200 = 1800元

3. 股价在100元
   - 股票盈亏：0
   - 期权价值：0（虚值）
   - 净亏损：200元（保险费）
```

### 策略优缺点
```
优点：
✓ 下跌保护
✓ 保留上涨收益
✓ 心理安全感
✓ 灵活性（可选择行权价）

缺点：
✗ 权利金成本
✗ 降低收益率
✗ 频繁保险成本累积
```

### 行权价选择
```
低行权价（虚值）：
- 权利金便宜
- 保护程度低
- 类似"免赔额高"的保险

高行权价（实值）：
- 权利金贵
- 保护程度高
- 类似"全险"

建议：
- 通常选择虚值10-15%行权价
- 平衡成本与保护
```

## 备兑看涨（Covered Call）

### 策略构成
```
组合：持有股票 + 卖出看涨期权

收入：收取权利金

效果：增强收益，降低持仓成本
```

### 盈亏分析
```
最大收益：行权价 - 股票成本 + 权利金
最大损失：股票成本 - 权利金（股票归零）

盈亏平衡点：股票成本 - 权利金

图形：
利润
 │      ______
 │     /
 │____/
-│____________ 股票价格
     成本  行权价

特点：
- 上涨收益有限
- 下跌有缓冲（权利金）
- 适合震荡或温和上涨
```

### 应用示例
```
情景：
- 持有100股某股票，成本价100元
- 预期短期震荡或小幅上涨
- 卖出1张看涨期权（行权价105元，权利金3元）

结果分析：
1. 股价涨到110元（超行权价）
   - 股票被行权，卖出价105元
   - 收益：(105-100+3)×100 = +800元
   - 机会成本：错失110-105=5元

2. 股价涨到103元（未达行权价）
   - 期权无价值，保留权利金
   - 收益：(103-100+3)×100 = +600元
   - 比单纯持股多赚300元

3. 股价跌到95元
   - 股票亏损：(95-100)×100 = -500元
   - 期权权利金：+300元
   - 净亏损：200元（而非500元）
```

### 策略优缺点
```
优点：
✓ 增强收益（权利金）
✓ 降低持仓成本
✓ 震荡市有效
✓ 风险低于单纯持股

缺点：
✗ 上涨收益受限
✗ 股票被行权风险
✗ 不适合强烈看涨
```

### 行权价选择
```
低行权价（实值）：
- 权利金高
- 更容易被行权
- 适合保守投资者

高行权价（虚值）：
- 权利金低
- 保留更多上涨空间
- 适合温和看涨

建议：
- 通常选择虚值5-10%行权价
- 平衡收益增强与上涨空间
```

## 策略对比

| 维度 | 保护性看跌 | 备兑看涨 |
|------|------------|----------|
| 适用预期 | 担心下跌 | 震荡/温和上涨 |
| 构建方式 | 买股票+买put | 买股票+卖call |
| 现金流 | 支出 | 收入 |
| 最大收益 | 无限 | 有限 |
| 最大损失 | 有限 | 有限（大于备兑） |
| 主要目的 | 保险 | 增强收益 |

## 组合策略

### Collar（领口策略）
```
构成：
- 持有股票
- 买入看跌（保护下行）
- 卖出看涨（支付保护成本）

效果：
- 低成本保护
- 收益和风险都有限
- 适合保守投资者

示例：
- 持有股票100元
- 买入95 put（权利金2元）
- 卖出105 call（权利金2元）
- 净权利金：0
- 结果：95-105区间震荡锁定
```
