# 53. 窝轮与牛熊证 (Warrants & CBBCs)

## 窝轮（Warrant）

### 概念定义
窝轮是由金融机构发行的期权类衍生品，赋予持有者在特定日期或之前以特定价格买入（认购）或卖出（认沽）标的资产的权利。

### 窝轮 vs 期权
```
┌───────────┬──────────┬──────────┐
│   特征    │   窝轮   │   期权   │
├───────────┼──────────┼──────────┤
│ 发行方    │ 金融机构 │ 交易所   │
│ 交易场所  │ 交易所   │ 交易所   │
│ 对手方    │ 发行商   │ 市场     │
│ 行权      │ 通常现金 │ 实物/现金│
│ 条款      │ 标准化低 │ 标准化高│
│ 引伸波幅  │ 由发行商定│ 市场定价│
└───────────┴──────────┴──────────┘
```

### 窝轮命名规则
```
示例：腾讯摩通二四购A
├── 腾讯：标的资产
├── 摩通：发行商（摩根大通）
├── 二四：到期年份24年
├── 购：认购（Call）/沽：认沽（Put）
└── A：同一发行商同款第A只

代码示例：
- 12789.HK
- 可查具体条款
```

### 窝轮关键条款
```
1. 兑换比率
   - 多少份窝轮换1股正股
   - 常见10:1, 100:1
   - 影响杠杆

2. 行使价
   - 约定的买卖价格
   - 影响内在价值

3. 到期日
   - 最后交易日
   - 欧式（只到期行权）

4. 街货量
   - 市场持有量
   - 过高可能有操纵风险
```

### 窝轮特点
```
优点：
✓ 杠杆高，资金门槛低
✓ 可做对冲
✓ 交易成本相对低

缺点：
✗ 时间损耗快
✗ 引伸波幅变化风险
✗ 发行商报价质量差异
✗ 到期归零风险
```

## 牛熊证（CBBC - Callable Bull/Bear Contract）

### 概念定义
牛熊证是一种带有强制回收机制的杠杆投资工具，追踪标的资产表现，设有强制收回价（Knock-out Price）。

### 牛证 vs 熊证
```
牛证（Bull）：
- 看涨工具
- 收回价在现价下方
- 跌破收回价即强制收回

熊证（Bear）：
- 看跌工具
- 收回价在现价上方
- 升破收回价即强制收回
```

### 关键条款
```
1. 行使价
   - 到期结算价

2. 收回价
   - 触发强制收回的价格
   - N类：收回价=行使价（无剩余价值）
   - R类：收回价≠行使价（可能有剩余价值）

3. 财务费用
   - 发行商融资成本
   - 每日计提
   - 持有成本

4. 杠杆
   - 高杠杆（通常5-20倍）
```

### 牛熊证运作示例
```
牛证示例：
- 标的：腾讯，现价400元
- 牛证行使价：350元
- 收回价：360元（N类）
- 兑换比率：100:1
- 价格：0.50元

情景1：腾讯涨到450元
- 内在价值 = (450-350)/100 = 1.00元
- 盈利100%

情景2：腾讯跌到360元
- 触发强制收回
- N类：价值归零
- 损失100%
```

### 牛熊证特点
```
优点：
✓ 高杠杆
✓ 价格透明（与标的直接挂钩）
✓ 无引伸波幅影响
✓ 强制收回机制限制损失

缺点：
✗ 强制收回风险
✗ 财务费用消耗
✗ 不适合长期持有
✗ 高波动下易被收回
```

## 窝轮 vs 牛熊证

| 维度 | 窝轮 | 牛熊证 |
|------|------|--------|
| 风险 | 时间损耗 | 强制收回 |
| 杠杆 | 高 | 更高 |
| 引伸波幅 | 敏感 | 不敏感 |
| 适合 | 中短期方向 | 短期方向 |
| 最大损失 | 权利金 | 可能全部 |
| 财务费用 | 无 | 有 |

## 交易注意事项

### 窝轮选择
```
1. 选择活跃发行商
2. 街货量不要过高（<30%）
3. 到期时间不要太近（>1个月）
4. 引伸波幅合理
5. 实际杠杆适中
```

### 牛熊证选择
```
1. 收回价与现价距离适中
2. 注意财务费用
3. N类 vs R类选择
4. 避免高波动时入场
5. 严格止损
```
