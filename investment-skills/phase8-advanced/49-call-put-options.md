# 49. 看涨看跌期权 (Call and Put Options)

## 看涨期权（Call Option）详解

### 买方视角
```
权利：以行权价买入标的
成本：权利金
收益：上涨时获利
风险：最多损失权利金

盈亏分析：
盈亏 = max(0, 到期价格 - 行权价) - 权利金

盈亏平衡点 = 行权价 + 权利金

图形：
利润
 │      ╱
 │     ╱
 │____╱_____ 股票价格
-权利金  行权价
```

### 卖方视角
```
义务：被要求时以行权价卖出标的
收入：权利金
风险：理论上无限

盈亏分析：
盈亏 = 权利金 - max(0, 到期价格 - 行权价)

盈亏平衡点 = 行权价 + 权利金

最大收益：权利金
最大损失：理论上无限
```

### 买入看涨的应用场景
```
1. 杠杆做多
   - 资金有限但强烈看涨
   - 杠杆效应放大收益

2. 风险对冲
   - 持有现金等待时机
   - 用少量资金锁定买入价

3. 波动率交易
   - 预期波动率上升
   - 买入期权做多波动
```

## 看跌期权（Put Option）详解

### 买方视角
```
权利：以行权价卖出标的
成本：权利金
收益：下跌时获利
风险：最多损失权利金

盈亏分析：
盈亏 = max(0, 行权价 - 到期价格) - 权利金

盈亏平衡点 = 行权价 - 权利金

图形：
利润
 │╲
 │ ╲
 │__╲____ 股票价格
    行权价  行权价-权利金
-max(权利金)
```

### 卖方视角
```
义务：被要求时以行权价买入标的
收入：权利金
风险：行权价 - 0 - 权利金

盈亏分析：
盈亏 = 权利金 - max(0, 行权价 - 到期价格)

盈亏平衡点 = 行权价 - 权利金

最大收益：权利金
最大损失：行权价 - 权利金（股票跌到0）
```

### 买入看跌的应用场景
```
1. 做空工具
   - 看空某股票
   - 无需融券，风险可控

2. 保护性看跌
   - 持有股票买看跌
   - 保险策略，限制下跌风险

3. 波动率交易
   - 预期波动率上升
   - 买入看跌做多波动
```

## 期权策略矩阵

### 四种基本头寸
```
              看涨       看跌
买入      看涨买方    看跌买方
          (权利金↑)   (权利金↑)
          看涨        看跌

卖出      看涨卖方    看跌卖方
          (收权利金)  (收权利金)
          看跌/中性   看涨/中性
```

### 市场观点与策略选择
```
强烈看涨：
- 买入看涨（ leverage ）
- 卖出看跌（温和看涨）

温和看涨：
- 卖出看跌
- 牛市价差

中性：
- 卖出跨式（收权利金）
- 日历价差

温和看跌：
- 卖出看涨
- 熊市价差

强烈看跌：
- 买入看跌
- 卖出看涨（温和看跌）
```

## 期权买方 vs 卖方

| 维度 | 买方 | 卖方 |
|------|------|------|
| 胜率 | 较低（通常30-40%） | 较高（通常60-70%） |
| 盈亏比 | 高（以小博大） | 低（以大博小） |
| 资金要求 | 小（仅权利金） | 大（保证金） |
| 风险 | 有限（权利金） | 可能很大 |
| 收益 | 无限（买方） | 有限（权利金） |
| 时间价值 | 损耗 | 收益 |
| 适合 | 方向明确 | 方向不明/震荡 |
