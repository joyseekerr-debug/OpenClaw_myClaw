# 65. Smart Beta策略 (Smart Beta Strategies)

## 概念定义

Smart Beta（聪明贝塔）是介于主动管理和被动指数之间的投资策略，通过系统化的方法捕捉特定的风险因子溢价，而非简单的市值加权。

## Smart Beta vs 传统指数

### 传统市值加权指数
```
特点：
- 按市值加权
- 高价买入（涨多了权重高）
- 低价卖出（跌多了权重低）
- 机械被动

问题：
- 可能追高
- 包含所有股票（含劣质）
- 无法获取因子溢价
```

### Smart Beta指数
```
特点：
- 按特定规则加权
- 透明、系统化
- 低成本
- 目标：风险调整后的超额收益

方法：
- 基于因子选择/加权
- 等权重
- 风险加权
- 基本面加权
```

## 主要Smart Beta策略

### 1. 价值策略
```
方法：
- 选择低估值股票
- 常用指标：PE、PB、PS、股息率

逻辑：
- 价值股长期跑赢成长股
- 市场过度反应后的修复

代表产品：
- 沪深300价值
- 标普500价值
```

### 2. 低波动策略
```
方法：
- 选择波动率低的股票
- 历史波动率、Beta

逻辑：
- 低波动异象
- 风险调整后收益更高
- 防御性强

适用：
- 熊市
- 风险厌恶投资者
- 降低组合波动
```

### 3. 动量策略
```
方法：
- 选择近期表现好的股票
- 通常3-12个月收益

逻辑：
- 趋势延续
- 行为金融学解释
- 市场反应不足

注意：
- 短期反转需规避
- 需要定期再平衡
```

### 4. 质量策略
```
方法：
- 选择高质量公司
- 指标：ROE、盈利稳定性、低杠杆

逻辑：
- 优质公司长期跑赢
- 可持续竞争优势
- 风险较低

适用：
- 长期投资
- 核心持仓
```

### 5. 等权重策略
```
方法：
- 每只股票相同权重
- 定期再平衡

vs 市值加权：
- 小盘股权重更高
- 自动高抛低吸
- 交易成本低

代表：
- 标普500等权重
```

### 6. 风险加权策略
```
方法：
- 按风险（波动率）加权
- 低风险股票高权重

vs 等权重：
- 考虑风险差异
- 风险贡献相等

适用：
- 风险平价
- 组合构建
```

### 7. 基本面加权
```
方法：
- 按基本面指标加权
- 营收、利润、分红、账面价值

代表：
- 锐联基本面指数（RAFI）

逻辑：
- 按经济规模而非市值
- 避免估值泡沫
```

### 8. 多因子策略
```
方法：
- 组合多个因子
- 价值+质量+低波

优势：
- 分散因子风险
- 更稳定
- 适应不同市场

代表：
- MSCI多因子指数
- 自定义多因子
```

## Smart Beta优势

```
1. 透明
   - 规则明确
   - 无黑箱

2. 低成本
   - 费率低于主动基金
   - 高于普通ETF

3. 系统化
   - 避免人为情绪
   - 纪律执行

4. 因子暴露
   - 获取长期因子溢价
   - 风险调整收益

5. 灵活
   - 多种因子选择
   - 可定制
```

## Smart Beta风险

```
1. 因子周期性
   - 因子会失效
   - 有回撤期
   - 需要长期持有

2. 拥挤风险
   - 使用人多后衰减
   - Smart Beta不Smart

3. 再平衡成本
   - 定期调整
   - 交易成本
   - 跟踪误差

4. 历史偏差
   - 回测过拟合
   - 未来表现不确定
```

## Smart Beta选择

### 选择因素
```
1. 投资目标
   - 增强收益？
   - 降低风险？
   - 特定暴露？

2. 因子理解
   - 相信哪个因子？
   - 理解其逻辑？
   - 能承受周期？

3. 成本考量
   - 费用率
   - 跟踪误差
   - 流动性

4. 市场环境
   - 当前适合什么因子？
   - 估值水平
   - 经济周期
```

### 常用组合
```
保守型：低波 + 质量 + 价值
平衡型：价值 + 质量 + 动量
成长型：动量 + 质量
防御型：低波 + 高股息
```
