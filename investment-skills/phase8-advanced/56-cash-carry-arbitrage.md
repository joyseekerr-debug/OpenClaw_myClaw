# 56. 期现套利 (Cash and Carry Arbitrage)

## 概念定义

期现套利是利用期货价格与现货价格之间的偏离，通过同时建立期货和现货的反向头寸，待价差回归正常后平仓获利的无风险套利策略。

## 理论基础

### 期货定价公式
```
理论期货价格 = 现货价格 × e^((r + s - q) × T)

简化公式（线性）：
理论期货价格 = 现货价格 × (1 + (r + s - q) × T)

其中：
r = 无风险利率
s = 存储成本率（商品）
q = 股息率（股票）/便利收益率（商品）
T = 到期时间（年）
```

### 无套利区间
```
考虑交易成本后：

正向套利边界：
期货价格 > 现货价格 × (1 + C) + 成本

反向套利边界：
期货价格 < 现货价格 × (1 - C) - 成本

其中C为持有成本
```

## 套利类型

### 正向套利（Cash and Carry）

#### 适用条件
```
期货价格 > 现货价格 + 持有成本 + 交易成本

即：期货高估（Contango过深）
```

#### 操作流程
```
步骤1：
- 借入资金
- 买入现货

步骤2：
- 卖出等量期货

步骤3：
- 持有至到期
- 用现货交割期货

步骤4：
- 收回资金
- 偿还本息
- 剩余为利润

利润 = 期货价格 - 现货价格 - 持有成本
```

### 反向套利（Reverse Cash and Carry）

#### 适用条件
```
期货价格 < 现货价格 - 借券成本 - 交易成本

即：期货低估（Backwardation过深）
```

#### 操作流程
```
步骤1：
- 借入现货（融券）
- 卖出现货

步骤2：
- 买入等量期货

步骤3：
- 持有至到期
- 期货交割获得现货

步骤4：
- 归还借入的现货
- 支付借券费用
- 剩余资金为利润

利润 = 现货价格 - 期货价格 - 借券成本
```

## 股指期货期现套利

### 套利实施
```
现货选择：
- 直接买入指数成分股
- 买入ETF（常用）
- 买入指数基金

套利操作：
正向套利：
- 买入ETF（现货）
- 卖出股指期货
- 到期交割或提前平仓

反向套利（受限）：
- 融券卖出ETF
- 买入股指期货
- 受融券 availability 限制
```

### ETF选择
```
考虑因素：
1. 跟踪误差小
2. 流动性好
3. 交易成本低
4. 可融资融券

常用ETF：
- 沪深300ETF
- 上证50ETF
- 中证500ETF
```

### 成本计算
```
正向套利成本：
├── ETF交易成本（佣金+冲击）：0.1%
├── 期货交易成本：0.01%
├── 资金成本（年化5%×1/12）：0.4%
└── 其他费用：0.05%
合计：约0.56%（月度）

触发阈值：
- 年化升水 > 6-7%
- 月度升水 > 0.6%
```

## 商品期货期现套利

### 特点
```
与股指期货区别：
1. 有实物交割
2. 有存储成本
3. 有运输成本
4. 品质可能有差异
```

### 操作流程
```
正向套利：
- 买入现货商品
- 存储在指定仓库
- 卖出期货合约
- 到期交割

额外考虑：
- 仓储费用
- 保险费用
- 质检费用
- 运输费用
```

### 风险
```
1. 存储风险
   - 商品变质
   - 损耗

2. 资金风险
   - 资金占用大
   - 期货保证金追加

3. 交割风险
   - 质量标准
   - 交割流程
```

## 期现套利风险管理

### 基差风险
```
风险：
- 价差可能继续扩大
- 需要追加保证金

控制：
- 严格止损
- 控制仓位
- 预留充足资金
```

### 展期风险
```
情况：
- 近月合约到期
- 需要展期到远月
- 展期成本不确定

处理：
- 提前规划
- 滚动展期
- 监控展期成本
```

### 执行风险
```
风险：
- 无法同时成交
- 滑点损失

控制：
- 程序化交易
- 限价单
- 接受合理滑点
```

## 期现套利收益特征

```
收益特点：
- 低风险（理论上无风险）
- 低收益（年化5-15%）
- 需要大资金
- 机会不常有

适合投资者：
- 机构投资者
- 追求稳定收益
- 有资金优势
- 有技术系统
```
