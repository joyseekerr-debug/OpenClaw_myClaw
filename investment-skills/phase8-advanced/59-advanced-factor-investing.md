# 59. 因子投资进阶 (Advanced Factor Investing)

## 因子投资回顾

### 传统因子
```
价值因子（Value）：
- 低PE、低PB、高股息
- 长期超额收益来源

规模因子（Size）：
- 小市值效应
- 风险补偿

动量因子（Momentum）：
- 趋势延续
- 行为金融学解释

质量因子（Quality）：
- 高ROE、低杠杆、盈利稳定
- 防御性强
```

## 进阶因子

### 1. 低波动因子（Low Volatility）
```
定义：
- 选择波动率低的股票
- 历史波动率、Beta

悖论：
- 低波动股票收益反而高
- 与CAPM矛盾

解释：
- 杠杆限制
- 彩票偏好
- 机构行为

应用：
- 防御型组合
- 降低回撤
```

### 2. 流动性因子（Liquidity）
```
定义：
- 选择低流动性股票
- 换手率、成交量、Amihud指标

逻辑：
- 流动性差需要补偿
- 小盘股、冷门股

风险：
- 交易成本高
- 难以大规模实施
- 危机时流动性枯竭
```

### 3. 情绪因子（Sentiment）
```
定义：
- 利用市场情绪
- 分析师预期、媒体情绪

指标：
- 盈利惊喜
- 分析师一致预期变化
- 新闻情绪
- 期权隐含情绪

策略：
- 逆势情绪（高恐慌买入）
- 顺势情绪（趋势确认）
```

### 4. 宏观因子（Macro）
```
定义：
- 暴露于宏观经济变量
- 非股票特有因子

类型：
- 经济增长
- 通货膨胀
- 利率
- 汇率

应用：
- 跨资产配置
- 风险预算
- 宏观对冲
```

## 多因子模型进阶

### 因子相关性
```
目标：
- 选择低相关因子
- 提高分散效果

低相关组合示例：
- 价值 + 动量
- 质量 + 规模
- 低波 + 红利

避免高相关：
- 价值与红利（高相关）
- 动量与成长（高相关）
```

### 因子择时
```
方法1：宏观经济择时
- 经济扩张：价值、小盘
- 经济收缩：质量、低波

方法2：市场情绪择时
- 高情绪：动量
- 低情绪：价值

方法3：因子动量
- 近期表现好的因子继续配
- 因子层面的追涨杀跌

争议：
- 择时难度大
- 交易成本
- 风格漂移
```

### 因子风险预算
```
方法：
- 各因子风险贡献相等
- 而非资金分配相等

实施：
1. 估计因子收益协方差
2. 计算各因子风险贡献
3. 调整权重使贡献相等
4. 动态再平衡

优势：
- 真正的分散化
- 避免风险集中在某因子
```

## 因子投资实施优化

### 信号构建
```
单一指标 → 复合信号：
- 价值：PE+PB+PS+PCF综合
- 质量：ROE+盈利稳定性+杠杆
- 避免单一指标噪音

信号处理：
- 去极值（Winsorize）
- 标准化（Z-score）
- 中性化（行业/市值）
```

### 组合构建
```
优化方法：
- 均值方差优化
- 风险平价
- 约束优化（换手率、行业偏离）

约束条件：
- 行业中性
- 市值中性
- 换手率限制
- 个股权重上限
```

### 执行优化
```
再平衡频率：
- 月度、季度
- 平衡交易成本与信号衰减

交易执行：
- VWAP算法
- 分多日执行
- 减少市场冲击
```
