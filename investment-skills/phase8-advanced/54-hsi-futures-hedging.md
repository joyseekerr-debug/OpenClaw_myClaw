# 54. 恒指期货对冲 (Hang Seng Index Futures Hedging)

## 恒指期货基础

### 合约规格
```
标的：恒生指数
合约乘数：每点50港元
最小跳动：1点（=50港元）
交易时间：
- 早盘：09:15-12:00
- 午盘：13:00-16:30
- 夜盘：17:15-03:00（次日）
合约月份：当月、下月及季月
交割方式：现金结算
```

### 保证金
```
初始保证金：约12万港元/手（随市场调整）
维持保证金：约9万港元/手

杠杆效应：
- 恒指25000点时，合约价值 = 125万港元
- 保证金约10%，杠杆约10倍
```

## 对冲原理

### 为什么要对冲
```
1. 系统性风险
   - 持有股票组合
   - 担心大盘下跌
   - 不想卖出股票

2. 事件风险
   - 重要事件前（选举、政策）
   - 短期避险
   - 事后解除对冲

3. 锁定期货利
   - IPO锁定期
   - 无法卖出
   - 期货对冲风险
```

### 对冲比率计算
```
对冲比率 = 组合市值 / (恒指点数 × 合约乘数) × Beta

示例：
- 股票组合市值：500万港元
- 恒指当前：25000点
- 组合Beta：1.2

对冲比率 = 5,000,000 / (25,000 × 50) × 1.2
         = 5,000,000 / 1,250,000 × 1.2
         = 4 × 1.2
         = 4.8手

实际：卖空5手恒指期货
```

## 对冲策略

### 完全对冲
```
目标：完全消除市场风险

操作：
- 卖空与组合等值的期货
- Beta调整后的对冲比率

结果：
- 大盘涨跌不影响组合价值
- 只承担个股风险
- 对冲成本：保证金占用、基差风险
```

### 部分对冲
```
目标：降低风险敞口

操作：
- 对冲部分市值（如50%）

适用：
- 不确定市场方向
- 降低波动但保留部分上涨

示例：
- 对冲50%市值
- 恒指涨10%：组合涨约5%
- 恒指跌10%：组合跌约5%
```

### 动态对冲
```
目标：持续调整对冲比率

调整触发：
- 组合市值变化较大
- Beta发生变化
- 期货到期移仓

注意：
- 交易成本
- 调整频率
- 跟踪误差
```

## 对冲风险

### 基差风险
```
基差 = 现货价格 - 期货价格

正常情况（Contango）：
- 期货价格 > 现货价格
- 持有成本

反向情况（Backwardation）：
- 期货价格 < 现货价格
- 对冲可能有额外成本

基差变化会影响对冲效果
```

### 跟踪误差
```
来源：
- 组合与指数成分不同
- Beta估计误差
- 期货合约展期

影响：
- 对冲不完全
- _residual风险

降低方法：
- 使用多个期货
- 优化组合Beta
```

### 保证金风险
```
风险：
- 期货价格反向变动
- 需要追加保证金
- 强制平仓风险

管理：
- 预留充足资金
- 监控保证金比例
- 设置风险限额
```

## 对冲应用示例

### 情景1：保护持仓
```
背景：
- 持有500万港股组合
- 担心短期调整
- 不想卖出股票

操作：
- 恒指25000点
- 卖空4手恒指期货
- 占用保证金约48万

结果：
- 恒指跌5%（1250点）
- 期货盈利：1250×50×4 = 25万
- 组合亏约5%（25万）
- 净效果：对冲成功
```

### 情景2：IPO锁定期对冲
```
背景：
- 新股中签，锁定期3个月
- 担心期间市场下跌

操作：
- 上市首日卖空恒指期货
- 对冲市值敞口

注意：
- 个股与指数相关性
- 可能不完全对冲
- 考虑个股Beta
```

## 对冲解除

### 解除时机
```
1. 风险消除
   - 担忧事件过去
   - 市场企稳

2. 目标达成
   - 达到对冲目标
   - 获利了结

3. 成本考虑
   - 对冲成本过高
   - 改变策略
```

### 解除方式
```
1. 平仓期货
   - 买回期货合约
   - 最简单直接

2. 移仓
   - 平仓近月，开仓远月
   - 延续对冲

3. 部分解除
   - 减少对冲比率
   - 逐步恢复敞口
```
