# 58. ESG投资 (ESG Investing)

## 概念定义

ESG投资是指在投资决策中考虑环境（Environmental）、社会（Social）和治理（Governance）三个维度的投资方法。它超越了传统财务分析，关注企业的可持续发展能力和社会责任。

## ESG三大维度

### E - 环境（Environmental）
```
关注因素：
├── 气候变化
│   ├── 碳排放量
│   ├── 温室气体管理
│   └── 气候风险暴露
├── 自然资源
│   ├── 水资源管理
│   ├── 土地使用
│   └── 生物多样性
├── 污染与废弃物
│   ├── 空气污染
│   ├── 水污染
│   └── 废弃物处理
└── 环境机遇
    ├── 清洁技术
    ├── 可再生能源
    └── 绿色建筑
```

### S - 社会（Social）
```
关注因素：
├── 人力资本
│   ├── 员工福利
│   ├── 培训发展
│   ├── 健康安全
│   └──  diversity & inclusion
├── 产品责任
│   ├── 产品质量
│   ├── 数据安全
│   ├── 隐私保护
│   └── 负责任营销
├── 利益相关方
│   ├── 社区关系
│   ├── 供应链管理
│   └── 人权保护
└── 社会机遇
    ├── 金融服务普惠
    ├── 健康与营养
    └── 教育与就业
```

### G - 治理（Governance）
```
关注因素：
├── 公司治理
│   ├── 董事会结构
│   ├── 独立董事比例
│   ├── 审计独立性
│   └── 股东权利
├── 商业道德
│   ├── 反腐败
│   ├── 反不正当竞争
│   ├── 税务透明
│   └── 举报机制
└── 风险管理
    ├── 风险管理体系
    ├── 内部控制
    └── 合规文化
```

## ESG投资方法

### 1. ESG整合（ESG Integration）
```
方法：
- 将ESG因素纳入传统财务分析
- 评估ESG对风险和收益的影响
- 调整估值模型

示例：
- 高碳企业：考虑碳成本，下调估值
- 治理差的企业：增加风险溢价
```

### 2. 负面筛选（Negative Screening）
```
方法：
- 排除不符合ESG标准的公司/行业

常见排除：
- 烟草
- 武器
- 煤炭
- 争议性武器
- 严重人权问题

适用：
- 价值观驱动的投资者
- 宗教机构
- 公益基金
```

### 3. 正面筛选（Positive Screening）
```
方法：
- 选择ESG表现优秀的公司
- 投资行业领先的ESG企业

类型：
- Best-in-class：各行业选ESG领先者
- 主题投资：清洁能源、可持续农业等
```

### 4. 影响力投资（Impact Investing）
```
方法：
- 主动寻求可衡量的社会/环境影响
- 同时获得财务回报

领域：
- 清洁能源
- 可负担住房
- 普惠金融
- 可持续农业

特点：
- 双重目标（财务+影响）
- 影响力测量
```

## ESG投资表现

### 研究结论
```
多数研究表明：
✓ ESG投资不会牺牲收益
✓ ESG优秀公司风险更低
✓ ESG可带来超额收益（部分研究）

原因：
- ESG优秀公司管理更好
- 风险识别更充分
- 长期竞争力强
- 监管风险低
```

### 争议
```
质疑观点：
✗ 增加成本
✗ 限制投资机会
✗ ESG评级不一致
✗ "漂绿"风险

回应：
- 长期看不增加成本
- 风险管理角度必要
- 评级标准在完善
- 监管加强打击漂绿
```

## ESG评级体系

### 主要评级机构
```
国际：
- MSCI ESG Ratings
- Sustainalytics
- FTSE Russell
- CDP

国内：
- 商道融绿
- 中证/华证ESG评级
-  Wind ESG评级
- 秩鼎技术
```

### 评级差异
```
问题：
- 不同机构评级相关性低
- 标准不统一
- 数据质量参差

原因：
- ESG数据披露不完整
- 权重设置不同
- 评价维度差异

建议：
- 参考多家评级
- 关注评级方法论
- 结合自主研究
```

## ESG投资实施

### 个人投资者
```
方式1：ESG基金
- ESG主题公募基金
- 跟踪ESG指数

方式2：个股筛选
- 查看公司ESG报告
- 关注负面新闻
- 参考ESG评级

方式3：股东行动
- 投票参与治理
- 提出ESG相关议案
```

### 机构投资者
```
ESG整合流程：
1. 制定ESG投资政策
2. 建立ESG研究框架
3. 整合到投资决策
4. 主动参与公司治理
5. 定期报告ESG表现

尽责管理：
- 与被投企业沟通
- 参与股东大会
- 协同其他投资者
```
