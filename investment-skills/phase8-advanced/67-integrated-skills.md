## 综合投资决策流程

### 阶段1：市场分析
```
宏观分析：
├── 经济周期位置
├── 货币政策走向
├── 地缘政治风险
└── 市场情绪指标

资产配置：
├── 股债商品比例
├── 地域分布
├── 行业配置
└── 风险预算分配
```

### 阶段2：标的筛选
```
量化筛选：
├── 财务质量过滤
├── 估值水平筛选
├── 技术形态过滤
└── 风险指标过滤

基本面验证：
├── 行业前景
├── 竞争优势
├── 管理层质量
└── 成长可持续性
```

### 阶段3：风险管理
```
入场前：
├── 仓位计算（凯利公式/固定比例）
├── 止损位设定
├── 风险敞口评估
└── 相关性检查

持仓中：
├── 实时监控
├── 移动止损
├── 再平衡触发
└── 风险预算管理
```

### 阶段4：执行交易
```
订单管理：
├── 订单类型选择
├── 时机选择
├── 分批执行
└── 滑点控制

成本控制：
├── 冲击成本管理
├── 算法交易
└── 最优执行
```

### 阶段5：监控与复盘
```
实时监控：
├── 价格/成交量监控
├── 风险指标监控
├── 预警响应
└── 异常处理

定期复盘：
├── 日度执行检查
├── 周度统计分析
├── 月度系统评估
└── 持续改进
```

## 不同市场环境策略组合

### 牛市环境
```
策略重点：
├── 趋势跟踪为主
├── 金字塔加仓
├── 移动止损保护利润
├── 适度杠杆放大收益

配置：
├── 股票：70-80%
├── 行业：成长+周期
├── 工具：股票+期权Call
```

### 熊市环境
```
策略重点：
├── 防御为主，降低仓位
├── 价值投资，逆向布局
├── 严格止损，保护本金
├── 对冲保护（Put/期货）

配置：
├── 股票：30-50%
├── 行业：防御性行业
├── 工具：债券+现金+Put
```

### 震荡市场
```
策略重点：
├── 均值回归策略
├── 高抛低吸
├── 区间交易
├── 卖期权收权利金

配置：
├── 股票：50-60%
├── 策略：多空平衡
├── 工具：期权策略（卖Straddle）
```

## 投资检查清单（综合版）

### 买入前检查
```
基本面：
□ 行业前景良好
□ 公司竞争优势明确
□ 财务状况健康
□ 估值合理

技术面：
□ 趋势方向明确
□ 入场点位合适
□ 成交量配合

风险管理：
□ 仓位比例合理
□ 止损位明确
□ 单笔风险可控
□ 组合相关性低

心理检查：
□ 无FOMO情绪
□ 非冲动决策
□ 独立思考
□ 接受最坏情况
```

### 持仓中检查
```
每日：
□ 止损位是否触发
□ 是否有重大消息
□ 情绪是否稳定
□ 是否按计划执行

每周：
□ 组合风险敞口
□ 收益/回撤情况
□ 是否需要再平衡
□ 策略有效性
```

### 卖出后检查
```
执行：
□ 是否按计划卖出
□ 卖出原因记录
□ 实际收益计算

复盘：
□ 决策质量评估
□ 改进点识别
□ 经验总结
```

## 投资成功法则

```
6. 持续学习
7. 控制情绪
8. 长期视角
```
