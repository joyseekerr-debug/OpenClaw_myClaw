# 62. 舆情分析 (Sentiment Analysis)

## 概念定义

舆情分析是指通过自然语言处理技术，对文本数据（新闻、社交媒体、研报等）进行情感倾向识别和主题提取，从而判断市场情绪、预测市场走势的分析方法。

## 舆情数据来源

### 1. 财经新闻
```
来源：
├── 主流媒体
│   ├── 彭博、路透
│   ├── 财新、第一财经
│   └── 华尔街日报、FT
├── 财经网站
│   ├── 东方财富
│   ├── 雪球
│   └── Seeking Alpha
└── 新闻通讯社
    ├── 新华社
    ├── 美联社
    └── 法新社

特点：
- 权威性强
- 时效性好
- 专业术语多
```

### 2. 社交媒体
```
平台：
├── Twitter/X
│   ├── 实时性强
│   ├── 信息量巨大
│   └── 噪音高
├── Reddit
│   ├── WallStreetBets
│   └── 散户情绪集中地
├── 国内平台
│   ├── 雪球（投资者社区）
│   ├── 东方财富股吧
│   └── 微信公众号
└── 专业论坛
    ├── 知乎
    └── 各券商论坛

特点：
- 反应迅速
- 情绪波动大
- 需去噪处理
```

### 3. 研究报告
```
类型：
├── 券商研报
│   ├── 评级变化
│   ├── 目标价调整
│   └── 业绩预测
├── 会议纪要
│   ├── 业绩说明会
│   ├── 投资者交流
│   └── 电话会议
└── 行业报告
    ├── 行业展望
    ├── 政策解读
    └── 专题研究

特点：
- 专业深度
- 影响持久
- 结构化程度高
```

## 舆情分析技术

### 1. 情感分析
```
方法：
├── 词典法
│   ├── 正向/负向词表
│   ├── 情感强度打分
│   └── 简单高效
├── 机器学习
│   ├── SVM、朴素贝叶斯
│   ├── 需要标注数据
│   └── 效果较好
└── 深度学习
    ├── LSTM、BERT
    ├── 上下文理解
    └── 效果最好但复杂

输出：
- 情感极性（正/负/中性）
- 情感强度（0-1）
- 情感变化趋势
```

### 2. 主题提取
```
方法：
├── TF-IDF
│   └── 关键词提取
├── LDA
│   └── 主题模型
├── 命名实体识别
│   └── 人名、公司名、产品名
└── 事件检测
    └── 突发事件识别

应用：
- 热点追踪
- 事件分类
- 关联分析
```

### 3. 关系抽取
```
内容：
- 公司与产品关系
- 人物与职位关系
- 事件与影响关系

应用：
- 知识图谱构建
- 产业链分析
- 因果推断
```

## 舆情量化指标

### 1. 情绪指数
```
计算方法：
- 正向文章数 / 总文章数
- 加权平均情感得分
- 与市场基准比较

常用指数：
- 恐慌/贪婪指数
- 投资者情绪指数
- 媒体情绪指数
```

### 2. 关注度指标
```
指标：
- 搜索量（百度指数、Google Trends）
- 文章数量
- 讨论热度
- 阅读量/转发量

应用：
- 热点发现
- 关注度拐点
- 反向指标（过度关注）
```

### 3. 预期差指标
```
方法：
- 分析师预期 vs 实际
- 市场情绪 vs 价格
- 正面舆情 vs 走势背离

应用：
- 业绩超预期预测
- 反转信号
- 预期修正交易
```

## 舆情交易策略

### 1. 事件驱动策略
```
触发：
- 重大新闻发布
- 业绩公告
- 政策出台
- 并购消息

执行：
- 快速解析情感
- 判断影响方向
- 自动/半自动交易

挑战：
- 速度要求高
- 真假辨别
- 市场反应不确定
```

### 2. 情绪反转策略
```
逻辑：
- 极端情绪往往反转
- 过度乐观 → 下跌
- 过度恐慌 → 上涨

指标：
- 情绪指数极值
- 多空情绪对比
- 情绪动量

应用：
- 逃顶
- 抄底
- 波段操作
```

### 3. 主题轮动策略
```
方法：
- 监测热点主题
- 追踪资金流向
- 提前布局

数据源：
- 政策文本分析
- 行业新闻热度
- 概念提及频率

风险：
- 热点切换快
- 追涨风险
- 需要快速响应
```

## 舆情分析挑战

### 数据挑战
```
质量问题：
- 虚假信息
- 水军刷量
-  sarcasm（讽刺）

处理：
- 来源可信度评估
- 去重去噪
- 多源验证
```

### 技术挑战
```
理解难度：
- 金融专业术语
- 隐含语义
- 上下文依赖

解决：
- 领域知识库
- 预训练金融模型
- 持续学习
```

### 市场挑战
```
信号衰减：
- 使用机构增多
- 市场效率提高
- 需要持续创新

噪音交易：
- 情绪与市场背离
- 基本面主导时失效
- 需要结合其他信号
```
