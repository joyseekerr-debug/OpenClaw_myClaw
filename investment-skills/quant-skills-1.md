## 157. SVM支持向量机

### 概念定义
支持向量机（Support Vector Machine）是一种监督学习算法，通过寻找最优超平面将数据分为不同类别，适用于分类和回归问题。

### 原理
- **最大间隔原则**：寻找使分类间隔最大的决策边界
- **支持向量**：决定超平面的关键样本点
- **核技巧**：通过核函数将低维数据映射到高维空间，解决非线性问题

### 常用核函数
1. **线性核**：适用于线性可分问题
2. **多项式核**：捕捉多项式关系
3. **RBF径向基核**：通用性强，适用于大多数非线性问题
4. **Sigmoid核**：模拟神经网络行为

### 在金融中的应用
- **涨跌预测**：基于技术指标预测次日涨跌方向
- **趋势分类**：识别上涨、下跌、震荡三种市场状态
- **信用评分**：评估债券违约风险
- **配对交易**：判断价差回归方向

### 优缺点
**优点**：
- 泛化能力强，不易过拟合
- 高维数据处理效果好
- 理论基础扎实

**缺点**：
- 大规模数据训练速度慢
- 核函数和参数选择需调优
- 对噪声数据敏感

### 注意事项
- 金融数据非平稳，需做适当预处理
- 类别不平衡问题需用SMOTE等方法处理
- 定期重新训练以适应市场变化
- 特征工程比算法选择更重要

## 158. 随机森林算法

### 概念定义
随机森林是一种集成学习方法，通过构建多棵决策树并汇总其预测结果，提高模型的准确性和稳定性。

### 原理
- **Bagging集成**：从训练集中有放回抽样构建多个子数据集
- **随机特征选择**：每棵树只使用随机选取的部分特征
- **投票机制**：分类问题多数投票，回归问题取平均

### 算法特点
1. **并行化**：各树独立训练，易于并行
2. **抗过拟合**：通过平均降低方差
3. **特征重要性**：可输出各特征的重要性评分
4. **异常检测**：可识别离群样本

### 金融应用场景
- **选股因子**：识别有效因子的重要性排序
- **违约预测**：信用风险评估
- **波动率预测**：集成多因子预测未来波动
- **策略集成**：组合多个子策略信号

### 关键参数
- **n_estimators**：树的数量，通常100-500
- **max_depth**：树的最大深度，控制复杂度
- **min_samples_split**：节点分裂最小样本数
- **max_features**：每次分裂考虑的特征数

### 注意事项
- 对高维稀疏数据效果一般
- 树数量过多边际效益递减
- 需进行交叉验证防止过拟合
- 特征重要性可能偏向高基数特征
- 预测结果难以解释，属于黑盒模型

## 159. KNN最近邻算法

### 概念定义
K近邻（K-Nearest Neighbors）是一种基于实例的学习方法，通过寻找与待预测样本最相似的K个样本进行预测。

### 原理
- **相似度度量**：常用欧氏距离、曼哈顿距离、马氏距离
- **K值选择**：决定决策边界的平滑程度
- **加权投票**：可根据距离远近赋予不同权重

### 距离度量选择
1. **欧氏距离**：适用于连续数值特征
2. **曼哈顿距离**：对异常值更鲁棒
3. **马氏距离**：考虑特征间相关性
4. **余弦相似度**：适用于方向比大小更重要的场景

### 金融应用
- **相似股票发现**：找到走势相似的股票对
- **市场状态识别**：判断当前市场类似历史哪个阶段
- **异常检测**：远离K个邻居的样本视为异常
- **因子选股**：选取与历史强势股票特征相似的股票

### 优缺点
**优点**：
- 简单直观，无需训练
- 非参数方法，适用性强
- 可增量学习

**缺点**：
- 预测时计算量大
- 对高维数据效果差（维度灾难）
- 对特征缩放敏感

### 注意事项
- 必须进行特征标准化
- K值选择需通过交叉验证
- 高维金融数据建议先降维
- 样本不平衡时需调整权重
- 历史相似不代表未来表现相同

## 160. 朴素贝叶斯

### 概念定义
朴素贝叶斯是基于贝叶斯定理的分类算法，假设特征之间相互独立，计算简单高效。

### 原理
- **贝叶斯定理**：P(A|B) = P(B|A) × P(A) / P(B)
- **朴素假设**：各特征条件独立（现实中很少成立，但效果往往不错）
- **最大后验概率**：选择使后验概率最大的类别

### 三种变体
1. **高斯朴素贝叶斯**：假设特征服从正态分布，适用于连续值
2. **多项式朴素贝叶斯**：适用于离散计数特征，如文本词频
3. **伯努利朴素贝叶斯**：适用于二元特征

### 金融应用
- **新闻分类**：自动分类财经新闻的利好/利空属性
- **情绪分析**：判断市场情绪的正面/负面倾向
- **涨跌预测**：基于历史特征概率预测涨跌
- **风险预警**：多指标组合的风险概率评估

### 优缺点
**优点**：
- 训练速度极快
- 对小样本数据效果好
- 天然支持多分类
- 对缺失值不敏感

**缺点**：
- 特征独立性假设过于强烈
- 概率估计可能不准确
- 对连续值处理较粗糙

### 注意事项
- 连续特征应做离散化处理
- 平滑技术（拉普拉斯平滑）避免零概率
- 对数变换防止概率下溢
- 特征相关性高时考虑其他算法
- 后验概率可作为置信度参考

## 161. 逻辑回归

### 概念定义
逻辑回归是一种广义的线性模型，通过Sigmoid函数将线性组合映射到(0,1)区间，用于二分类问题。

### 原理
- **Sigmoid函数**：f(x) = 1 / (1 + e^(-x))，输出可解释为概率
- **对数几率**：log(P/(1-P)) = β₀ + β₁X₁ + ... + βₙXₙ
- **最大似然估计**：通过优化算法求解最优参数

### 正则化方法
1. **L1正则（Lasso）**：产生稀疏解，自动特征选择
2. **L2正则（Ridge）**：防止过拟合，系数平滑
3. **弹性网络**：L1和L2的组合

### 金融应用
- **违约概率**：PD（Probability of Default）计算
- **涨跌预测**：次日上涨概率估计
- **因子筛选**：识别显著有效的选股因子
- **组合优化**：资产权重的概率约束

### 模型评估
- **混淆矩阵**：准确率、精确率、召回率、F1分数
- **ROC曲线**：AUC值衡量区分能力
- **KS统计量**：评分模型的区分度指标
- **对数损失**：概率预测的校准程度

### 注意事项
- 特征需做标准化处理
- 多重共线性问题需处理
- 样本不平衡需调整权重或采样
- 线性可分假设可能不满足
- 概率输出需校准（Calibration）
- 特征工程对效果影响巨大
