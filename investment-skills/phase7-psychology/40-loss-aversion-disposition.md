# 40. 损失厌恶与处置效应 (Loss Aversion & Disposition Effect)

## 损失厌恶（Loss Aversion）

### 概念定义

损失厌恶是前景理论的核心发现，指人们对损失的敏感度远高于同等收益，损失带来的痛苦约为同等收益带来快乐的2-2.5倍。

### 数学表达
```
价值函数特征：
V(x) = x^α      当 x ≥ 0（收益区，α≈0.88）
V(x) = -λ(-x)^β 当 x < 0（损失区，β≈0.88, λ≈2.25）

λ ≈ 2.25 表示损失厌恶系数
即：损失1元的痛苦 ≈ 获得2.25元的快乐
```

### 投资表现
```
1. 过早止盈
   - 盈利时风险厌恶
   - "落袋为安"心理
   - 错失后续涨幅

2. 过晚止损
   - 亏损时风险偏好
   - "只要不卖就不算亏"
   - 小亏变大亏

3. 盈亏不对称处理
   - 盈利股票：平均持有100天
   - 亏损股票：平均持有200天+
   - 结果：及时止盈，拖延止损

4. 无效交易
   - 卖出盈利股买亏损股
   - 试图"摊平成本"
   - 导致劣币驱逐良币
```

### 实际案例
```
场景：持有两只股票
- 股票A：成本100元，现价120元（盈利20%）
- 股票B：成本100元，现价80元（亏损20%）

常见决策：
1. 卖出股票A锁定盈利 ✅
2. 继续持有股票B等待回本 ❌

结果：
- 股票A继续上涨到150元（错失30%）
- 股票B继续下跌到60元（多亏20%）

理性决策应是：
- 评估当前价值而非成本
- 卖出被高估的，持有被低估的
- 与成本无关
```

### 克服损失厌恶
```
1. 心理账户分离
   - 每笔交易独立评估
   - 忘记成本价
   - 问：如果现在空仓，会买吗？

2. 重新定义参考点
   - 不以买入价为参考
   - 以当前价值为参考
   - 以投资组合整体为参考

3. 机械止损
   - 预设止损点
   - 自动执行
   - 避免临场犹豫

4. 长期视角
   - 单笔交易不重要
   - 关注长期期望收益
   - 接受亏损是成本
```

## 处置效应（Disposition Effect）

### 概念定义

处置效应是指投资者倾向于过早卖出盈利资产、过长时间持有亏损资产的现象。这是损失厌恶的直接表现。

### 实证研究
```
经典研究（Odean, 1998）：
- 分析了10,000个交易账户
- 发现：
  - 盈利股票出售概率：约15%
  - 亏损股票出售概率：约10%
  - 出售盈利股是亏损股的1.5倍

后续研究验证：
- 在多个市场、多类投资者中存在
- 即使专业投资者也难以避免
- 导致显著的投资收益损失
```

### 处置效应的成本
```
直接成本：
- 实现资本利得，提前缴税
- 错失盈利股的后续涨幅
- 承担亏损股的进一步下跌

机会成本：
- 资金困在亏损股
- 无法投资更好机会
- 组合整体收益下降

计算示例：
假设：
- 组合：50%盈利股（预期收益+15%）
-       50%亏损股（预期收益+5%）

处置效应决策：
- 卖出盈利股 → 实现+15%
- 持有亏损股 → 实际+5%
- 组合收益：+10%

理性决策：
- 卖出被高估的，持有被低估的
- 假设反过来操作：
- 持有优质股 → +15%
- 卖出劣质股 → 避免更大损失
- 组合收益可能更高
```

### 处置效应的心理机制
```
1. 实现收益的快乐
   - 卖出盈利时多巴胺分泌
   - 强化卖出行为
   - 追求即时满足

2. 避免后悔
   - 卖出亏损 = 承认错误
   - 延迟卖出 = 保留希望
   - 自我欺骗

3. 侥幸心理
   - "会涨回来的"
   - "只要回本我就卖"
   - 锚定成本价

4. 认知失调
   - 买入时的看好 vs 现在的亏损
   - 不一致带来心理不适
   - 通过"不卖"来缓解
```

### 克服处置效应
```
1. 逆向思维
   - 考虑：盈利股可能才是该留的
   - 亏损股可能才是该卖的
   - 让利润奔跑，快速止损

2. 定期再平衡
   - 不按盈亏卖出
   - 按配置比例卖出
   - 强制克服偏差

3. 卖出规则前置
   - 买入前设定卖出条件
   - 基于价值而非成本
   - 机械执行

4. 组合视角
   - 关注整体组合表现
   - 不纠结单只股票
   - 止损是为了组合健康
```

## 损失厌恶与处置效应的关系

```
损失厌恶（心理基础）
    ↓
处置效应（行为表现）
    ↓
投资绩效下降（结果）

打破链条的方法：
1. 意识：认识这些偏差
2. 系统：建立规则系统
3. 执行：机械化执行
4. 复盘：持续改进
```
