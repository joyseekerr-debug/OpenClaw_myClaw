# 41. 锚定效应与过度自信 (Anchoring & Overconfidence)

## 锚定效应（Anchoring Effect）

### 概念定义

锚定效应是指人们在决策时过度依赖最先获得的信息（锚点），后续判断和调整围绕锚点进行，且调整通常不足。

### 锚定类型

#### 1. 价格锚定
```
成本价锚定（最常见）：
- 买入价成为心理锚点
- 影响卖出决策
- "回到成本我就卖"
- "都跌这么多了还能跌多少"

案例：
100元买入股票：
- 涨到110元："赚了10%，考虑卖"
- 跌到90元："等回到100元就卖"
- 跌到70元："已经跌了30%，反弹在即"

问题：
- 决策基于成本而非价值
- 70元时可能高估
- 110元时可能低估

正确做法：
- 忘记成本价
- 评估当前价值
- 基于未来预期决策
```

#### 2. 历史价格锚定
```
前期高点锚定：
- 历史最高价成为阻力位
- "曾经到过100元"
- 影响对当前价格的判断

前期低点锚定：
- 历史最低价成为支撑位
- "最低也就到过50元"
- 可能低估进一步下跌风险

案例：
某股票历史最高200元，现100元：
- 锚定效应："已经腰斩，便宜"
- 可能忽视基本面恶化
- 实际可能只值50元
```

#### 3. 外部信息锚定
```
分析师目标价：
- 成为心理预期
- 到达后不管价值就卖
- 未到达就一直持有

整数关口：
- 100元、50元等
- 心理阻力/支撑
- 实际无技术意义

新闻数字：
- 报道中的特定数字
- 影响后续判断范围
```

### 克服锚定效应
```
1. 识别锚点
   - 意识到自己的锚点
   - 成本价、历史价、目标价
   - 问自己：如果今天才看到这只股票，会怎么做？

2. 多角度估值
   - 使用多种估值方法
   - PE、PB、DCF、相对估值
   - 不依赖单一数字

3. 情景分析
   - 乐观、中性、悲观情景
   - 不基于当前价格
   - 基于内在价值

4. 第二意见
   - 寻求外部观点
   - 不看成本的建议
   - 独立评估
```

## 过度自信（Overconfidence）

### 概念定义

过度自信是指高估自己的知识、能力、信息的准确性以及控制事件的能力。在投资中表现为过度交易、集中持仓和风险低估。

### 过度自信的三种形式

#### 1. 过度乐观（Over-optimism）
```
表现：
- 高估成功概率
- 低估失败风险
- 对未来过于看好

投资表现：
- 低估下跌风险
- 高估收益预期
- 忽视尾部风险

案例：
- "这次不一样"
- "我找到了十倍股"
- 不考虑最坏情况
```

#### 2. 过度精准（Over-precision）
```
表现：
- 过度确定自己的判断
- 置信区间过窄
- 忽视不确定性

投资表现：
- 重仓单一标的
- 杠杆过高
- 不做对冲

实验：
给出一个范围，让90%置信区间包含真实值
大多数人给出的范围过窄
即：过于相信自己知道准确数字
```

#### 3. 错觉控制（Illusion of Control）
```
表现：
- 相信自己能控制随机事件
- 技能与运气混淆
- 行动错觉

投资表现：
- 频繁交易寻求控制感
- 认为择时能显著提高收益
- 技术分析过度自信

案例：
- 抛硬币前用力吹一下
- 自己选择彩票号码
- 盯盘能带来控制感
```

### 过度自信的后果
```
1. 过度交易
   - 交易频率过高
   - 交易成本侵蚀收益
   - 研究发现：换手率越高，收益越低

2. 集中投资
   - 重仓少数标的
   - 缺乏分散化
   - 黑天鹅风险

3. 风险低估
   - 杠杆使用过度
   - 忽视尾部风险
   - 准备不足

4. 学习障碍
   - 归因偏差（成功归能力，失败归运气）
   - 难以从错误学习
   - 重复同样错误
```

### 过度自信的数据证据
```
Barber和Odean研究：
- 分析66,465个交易账户
- 发现：
  - 交易最频繁的20%账户
  - 年化收益比市场平均低约7%
- 原因：过度交易导致成本增加

性别差异：
- 男性比女性交易更频繁
- 单身男性最频繁
- 支持过度自信解释
```

### 克服过度自信
```
1. 记录预测
   - 记录每笔交易的预期
   - 事后检验准确率
   - 客观认识预测能力

2. 外部视角
   - 询问他人意见
   - 寻找反面观点
   - 考虑基础概率

3. 分散投资
   - 强制分散
   - 单只标的仓位限制
   - 避免过度集中

4. 降低交易频率
   - 设定交易间隔
   - 买入后锁定观察期
   - 减少冲动交易

5. 情景规划
   - 考虑最坏情况
   - 压力测试
   - 应急预案
```

## 锚定与过度自信的交互
```
常见组合：
1. 成本价锚定 + 过度自信
   - "我相信会回本"
   - 过度相信自己的判断
   - 迟迟不止损

2. 历史高点锚定 + 过度乐观
   - "曾经到过100元"
   - "这次一定能创新高"
   - 追高被套

应对：
- 意识到这种组合风险
- 建立系统性的决策流程
- 引入外部检查点
```
