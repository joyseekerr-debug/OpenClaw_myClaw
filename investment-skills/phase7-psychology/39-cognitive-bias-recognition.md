# 39. 认知偏差识别 (Cognitive Bias Recognition)

## 概念定义

认知偏差是指人类在信息处理和决策过程中系统性偏离理性的思维模式。在投资中，这些偏差会导致错误的判断和决策。

## 主要认知偏差

### 1. 确认偏误（Confirmation Bias）
```
定义：
倾向于寻找、解读和记忆支持自己已有观点的信息，
而忽视或贬低相反证据。

投资表现：
├── 买入后只看到利好
├── 忽视利空信号
├── 选择性阅读分析报告
└── 与同观点者交流更多

案例：
看好某股票后：
- 看到利好消息 → "果然如此"
- 看到利空消息 → "短期影响"
- 股价下跌 → "正是加仓机会"

克服方法：
✓ 主动寻找反面观点
✓ 设置"魔鬼代言人"
✓ 要求看空报告
✓ 定期检视假设
```

### 2. 可得性偏差（Availability Bias）
```
定义：
依据信息获取的容易程度来判断事件概率，
而非客观概率。

投资表现：
├── 最近亏损后过度悲观
├── 最近盈利后过度乐观
├── 媒体报道影响判断
├── 亲身经历权重过高

案例：
- 刚经历股灾 → 认为市场永远危险
- 周围人炒股赚钱 → 认为股市很容易
- 新闻报道航空事故 → 高估事故概率

克服方法：
✓ 使用长期统计数据
✓ 关注基础概率
✓ 建立检查清单
✓ 延迟决策
```

### 3. 代表性偏差（Representativeness Bias）
```
定义：
根据事物与典型模式的相似度做判断，
忽视基础概率和样本大小。

投资表现：
├── 看好某公司因为"像当年的苹果"
├── 根据小样本推断整体
├── 忽视回归均值
└── 模式识别过度

案例：
- 某股票连涨5天 → "趋势形成"
- 基金经理3年业绩好 → "能力出众"
- 忽视只是运气回归

克服方法：
✓ 关注样本大小
✓ 考虑基础概率
✓ 统计检验
✓ 区分信号与噪音
```

### 4. 锚定效应（Anchoring Bias）
```
定义：
过度依赖最先获得的信息（锚点），
后续判断围绕锚点调整不足。

投资表现：
├── 成本价锚定
├── 历史高点/低点锚定
├── 买入价成为卖出参考
└── 分析师目标价锚定

案例：
- 50元买的股票，跌到40元不想卖
  "已经跌了20%了"
- 100元的高点成为心理关口
- 第一报价影响后续判断

克服方法：
✓ 从当前价值重新评估
✓ 忽略成本价
✓ 考虑情景分析
✓ 多角度估值
```

### 5. 过度自信（Overconfidence）
```
定义：
高估自己的知识、能力和预测准确性。

投资表现：
├── 频繁交易
├── 集中持仓
├── 低估风险
├── 事后归因于能力

三种形式：
1. 高估能力："我能战胜市场"
2. 高估知识："我了解这家公司"
3. 高估控制："我能把握时机"

克服方法：
✓ 记录预测，检验准确率
✓ 分散投资
✓ 降低交易频率
✓ 寻求外部反馈
```

### 6. 后见之明偏误（Hindsight Bias）
```
定义：
事后认为事件是可预测的，
"我早就知道会这样"。

投资表现：
├── 错误中学习不足
├── 过度简化历史
├── 信心膨胀
└── 风险低估

案例：
- 危机后说"明显有泡沫"
- 忽视当时的不确定性
- 下次同样犯错

克服方法：
✓ 记录当时真实想法
✓ 承认不确定性
✓ 分析所有可能情景
✓ 谦虚面对未来
```

### 7. 框架效应（Framing Effect）
```
定义：
同一问题的不同表述影响决策，
即使实质相同。

投资表现：
├── "90%胜率" vs "10%失败率"
├── "止盈" vs "回撤控制"
├── 盈利和亏损的不同处理
└── 绝对收益 vs 相对收益

案例：
- 表述1："这只股票有70%可能涨20%"
  → 倾向于买入
- 表述2："这只股票有30%可能大跌"
  → 倾向于回避

克服方法：
✓ 转换视角思考
✓ 关注实质而非表述
✓ 使用数学期望
✓ 标准化决策框架
```

## 偏差识别自查清单

```
交易前检查：
□ 我是否只看了支持观点的信息？
□ 这个判断基于充分样本吗？
□ 我是不是被买入价影响了？
□ 我是否过度自信？
□ 如果换个表述，我还会这么做吗？

交易后检查：
□ 我是否事后诸葛亮？
□ 是否只记住了成功的预测？
□ 是否把运气当能力？
```
