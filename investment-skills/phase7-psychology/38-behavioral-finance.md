# 38. 行为金融学基础 (Behavioral Finance Fundamentals)

## 概念定义

行为金融学是将心理学理论与金融学相结合的学科，研究投资者的心理偏差如何影响投资决策和市场价格。它挑战了传统金融学的"理性人"假设，解释了市场中的非理性现象。

## 传统金融学 vs 行为金融学

### 对比
```
传统金融学（有效市场假说）：
├── 假设投资者完全理性
├── 能够无偏地处理信息
├── 以效用最大化为目标
├── 市场总是有效的
└── 价格始终反映内在价值

行为金融学：
├── 承认投资者存在认知偏差
├── 情绪和直觉影响决策
├── 有限理性，追求满意解
├── 市场并非总是有效
└── 价格可能长期偏离价值
```

### 有效市场的三种形式
```
弱式有效：
- 历史价格信息已反映
- 技术分析无效
- 行为金融学观点：可能有效

半强式有效：
- 公开信息已反映
- 基本面分析无效
- 行为金融学观点：部分有效

强式有效：
- 所有信息已反映
- 内幕信息也无法获利
- 行为金融学观点：不可能有效
```

## 行为金融学的核心概念

### 1. 有限理性（Bounded Rationality）
```
概念：
- 人类认知能力有限
- 无法处理全部信息
- 使用启发式简化决策

投资表现：
- 依赖简单规则而非复杂计算
- 关注显著信息忽略背景
- 使用心理捷径做判断
```

### 2. 前景理论（Prospect Theory）
```
卡尼曼和特沃斯基提出

核心观点：
├── 损失厌恶
│   └── 损失的痛苦 > 获得的快乐（约2:1）
├── 边际效用递减
│   └── 同样金额，贫穷时效用 > 富有时
├── 风险态度不对称
│   ├── 面对收益：风险厌恶
│   └── 面对损失：风险偏好
└── 参考点依赖
    └── 判断基于相对变化而非绝对值

价值函数特征：
        收益区（凹函数，风险厌恶）
       /
______/________ 参考点（原点）
     /
    / 损失区（凸函数，风险偏好）
```

### 3. 市场异象（Market Anomalies）
```
行为金融学解释的市场现象：

1. 动量效应
   - 过去涨的股票继续涨
   - 解释：反应不足、羊群效应

2. 反转效应
   - 长期看涨多了会跌
   - 解释：过度反应回归

3. 规模效应
   - 小盘股收益更高
   - 解释：被忽视、信息不透明

4. 价值效应
   - 低PE/PB股票收益更高
   - 解释：过度悲观后的修复

5. 日历效应
   - 一月效应、周末效应等
   - 解释：行为模式、资金流动
```

## 投资者心理偏差分类

### 认知偏差（Cognitive Biases）
```
信息处理偏差：
- 代表性偏差
- 可得性偏差
- 锚定效应
- 过度自信

判断偏差：
- 确认偏误
- 后见之明
- 框架效应
- 心理账户
```

### 情绪偏差（Emotional Biases）
```
恐惧与贪婪：
- 市场顶部贪婪
- 市场底部恐惧

后悔厌恶：
- 害怕后悔导致行动瘫痪
- 或冲动追悔

自我控制：
- 缺乏纪律
- 即时满足偏好
```

## 行为金融学应用

### 对投资的启示
```
1. 认识自己
   - 了解自身偏差
   - 识别情绪触发点

2. 建立系统
   - 用规则对抗情绪
   - 机械化执行

3. 逆向思考
   - 市场恐慌时寻找机会
   - 市场狂热时保持警惕

4. 长期视角
   - 短期市场非理性
   - 长期趋向价值
```

### 对市场的理解
```
1. 市场可能长期无效
   - 泡沫可以持续很久
   - 价值回归需要时间

2. 群体行为的预测性
   - 恐慌和贪婪会重复
   - 历史不会简单重复但会押韵

3. 套利限制
   - 即使识别错误定价
   - 套利也可能受限（有限套利）
```
