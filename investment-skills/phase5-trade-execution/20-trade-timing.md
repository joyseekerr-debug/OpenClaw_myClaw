# 20. 交易时机选择 (Trade Timing)

## 概念定义

交易时机选择是指在何时入场和离场的决策。好的时机选择可以降低交易成本、提高胜率，是交易执行的重要组成部分。

## 交易时段分析

### A股市场时段特征
```
09:15-09:25 集合竞价
├── 09:15-09:20 可撤单，试探性强
├── 09:20-09:25 不可撤单，真实意图
└── 09:25 撮合成交，确定开盘价

09:30-11:30 上午连续竞价
├── 09:30-10:00 开盘波动期
│   └── 隔夜信息释放，波动最大
├── 10:00-11:00 趋势形成期
│   └── 方向逐渐明朗
└── 11:00-11:30 上午收盘前
    └── 部分资金离场，波动减小

13:00-14:30 下午连续竞价
├── 13:00-13:30 午后开盘
│   └── 受午盘消息影响
├── 13:30-14:30 常规交易期
│   └── 全天最平稳时段

14:30-15:00 收盘前
├── 尾盘异动期
│   └── 机构调仓、ETF套利
└── 15:00 收盘集合竞价（深市）
```

### 美股市场时段特征
```
Pre-market (04:00-09:30)
├── 流动性较低
├── 波动较大
└── 适合应对盘前财报

Regular Hours (09:30-16:00)
├── 开盘(09:30-10:30)：波动最大
├── 午盘(11:00-14:00)：最清淡
└── 收盘(15:30-16:00)：成交量回升

After Hours (16:00-20:00)
├── 流动性进一步下降
├──  spreads扩大
└── 适合消化盘后消息
```

## 最佳交易时机

### 日内交易时机
```
高波动时段（适合突破策略）：
├── 开盘后30分钟
├── 重要数据发布时
├── 午盘消息发布时
└── 收盘前30分钟

低波动时段（适合均值回归）：
├── 午盘11:00-13:00
├── 长假前最后一个交易日
└── 重大事件前等待期
```

### 周内效应
```
周一：
├── 消化周末消息
├── 波动通常较大
└── 趋势可能延续或反转

周二-周四：
├── 最活跃的交易日
├── 流动性最好
└── 主要趋势形成期

周五：
├── 周末避险情绪
├── 尾盘可能减仓
└── 避免持仓过周末
```

### 月内效应
```
月初：
├── 新资金入场
├── 月初效应（Turn-of-the-month）
└── 机构调仓

月末/季末：
├── 资金紧张（银行间）
├── 调仓换股
└── 业绩考核影响
```

## 避开不良时机

### 避免交易时段
```
❌ 重大数据发布前5分钟
   - 波动剧烈，滑点巨大
   - 方向不明，赌博性质

❌ 长假前最后一个交易日尾盘
   - 流动性枯竭
   - 异常价格波动

❌ 连续涨跌停后开板
   - 情绪极端
   - 难以判断真实价值

❌ 自己状态不佳时
   - 疲劳、情绪激动
   - 违反纪律风险高
```

### 谨慎交易时段
```
⚠️ 开盘集合竞价
   - 价格发现过程
   - 可能大幅跳空

⚠️ 中午收盘前/下午开盘后
   - 方向可能改变
   - 等待趋势确认

⚠️ 重大会议/政策发布期间
   - 政策不确定性
   - 等待明朗后再行动
```

## 时机选择策略

### 趋势跟踪策略时机
```
入场时机：
├── 趋势确认后（突破、均线多头排列）
├── 回调至支撑位
└── 成交量配合放大

离场时机：
├── 趋势破坏信号
├── 达到目标位
└── 出现反向信号
```

### 均值回归策略时机
```
入场时机：
├── 过度偏离均线
├── 极值指标（RSI<30或>70）
└── 成交量萎缩

离场时机：
├── 回归均值
├── 趋势改变迹象
└── 达到时间限制
```

### 事件驱动策略时机
```
入场时机：
├── 事件确认后（如业绩超预期）
├── 事件影响未完全反应
└── 市场情绪稳定后

离场时机：
├── 事件影响充分消化
├── 下一个事件前
└── 达到预期收益
```

## 交易时机检查清单

```
□ 当前时段流动性是否充足？
□ 是否有重大数据/事件即将发布？
□ 当前波动率是否适合我的策略？
□ 我是否有足够时间监控这笔交易？
□ 我的心理状态是否适合交易？
□ 这笔交易是否在我的计划之内？
□ 是否考虑了隔夜/周末风险？
```

## 时机选择与交易成本

### 时间对成本的影响
```
理想时机：
- 流动性充足
- 买卖价差小
- 滑点可控
- 市场冲击小

不良时机成本：
- 价差扩大：额外0.1-0.5%成本
- 滑点增加：额外0.1-1%成本
- 市场冲击：大额订单影响价格
```
