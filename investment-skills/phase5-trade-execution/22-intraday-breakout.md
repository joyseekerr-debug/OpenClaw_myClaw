# 22. 盘中突破策略 (Intraday Breakout Strategy)

## 概念定义

盘中突破策略是指在交易日内，当价格突破关键价位（如前期高低点、整数关口、技术位等）时顺势入场的交易策略。它基于"突破后趋势延续"的假设。

## 突破类型

### 按时间框架分类
```
1. 日内突破
   - 突破当日开盘区间
   - 突破日内整理平台
   - 突破分时图高低点

2. 短期突破
   - 突破昨日高低点
   - 突破近期小平台

3. 中期突破
   - 突破周K线高低点
   - 突破重要均线
   - 突破整理形态（三角形、旗形等）
```

### 按突破方向分类
```
向上突破（做多）：
- 突破前期高点
- 突破阻力位
- 突破整理形态上轨

向下突破（做空）：
- 跌破前期低点
- 跌破支撑位
- 跌破整理形态下轨
```

## 突破确认要素

### 1. 价格确认
```
有效突破标准：
├── 收盘价突破（最可靠）
│   └── 当日收盘在关键价位之外
├── 连续两根K线突破
│   └── 避免单根K线假突破
├── 突破幅度≥1-2%
│   └── 足够幅度过滤噪音
└── 突破后回踩确认
    └── 原阻力变支撑
```

### 2. 成交量确认
```
突破成交量要求：
├── 突破时成交量放大
│   └── 至少为均量1.5倍以上
├── 放量突破后缩量回踩
│   └── 验证突破有效性
└── 对比历史突破成交量
    └── 参考成功突破的量价关系
```

### 3. 时间确认
```
突破时间验证：
├── 突破后维持在关键价位外的时间
│   └── 至少30分钟以上
├── 避免尾盘突破
│   └── 尾盘突破可靠性较低
└── 结合大盘时段特征
    └── 早盘突破优于午盘
```

## 盘中突破策略操作

### 突破买入策略
```
入场条件：
1. 价格突破前期高点/阻力位
2. 突破时成交量放大
3. 大盘环境配合（非单边下跌）
4. 板块有热点效应

入场方式：
- 突破瞬间市价追入
- 突破后回踩时买入
- 突破确认后分批建仓

止损设置：
- 跌破突破K线低点
- 跌破原阻力位（现支撑位）
- 固定百分比止损（如-3%）

止盈设置：
- 下一个技术阻力位
- 固定盈亏比（如2:1）
- 移动止盈跟踪
```

### 突破卖出策略
```
入场条件：
1. 价格跌破前期低点/支撑位
2. 跌破时成交量放大
3. 大盘走弱或板块回调
4. 无 immediate 支撑

注意事项：
- 下跌突破通常更急更快
- 止损要严格
- 仓位控制更严
```

## 突破策略的陷阱

### 假突破（False Breakout）
```
特征：
- 价格短暂突破后迅速回落
- 成交量未有效放大
- 多为诱多/诱空陷阱

识别：
- 突破后30分钟内回到原区间
- 突破K线长上影线/长下影线
- 成交量异常但价格不涨

应对：
- 等待收盘确认
- 设置严格止损
- 突破回踩确认后再入场
```

### 常见假突破场景
```
1. 早盘假突破
   - 开盘后情绪化交易
   - 10点前假突破率高

2. 整数关口假突破
   - 100元、50元等心理关口
   - 触及后迅速回落

3. 消息刺激假突破
   - 突发消息导致短暂突破
   - 消息消化后回归

4. 无量突破
   - 成交量未配合
   - 缺乏资金支撑
```

## 突破策略优化

### 多时间框架确认
```
日线突破 + 小时线确认：
- 日线级别突破信号
- 小时线等待回调入场
- 提高胜率，优化入场点

小时线突破 + 分钟线确认：
- 小时线级别突破
- 15分钟或5分钟确认
- 适合日内交易
```

### 多指标确认
```
突破 + 指标共振：
├── MACD金叉/死叉
├── RSI突破50中轴
├── 成交量均线突破
└── 布林带开口扩大
```

### 市场环境筛选
```
适合突破策略的市场：
✓ 趋势明确（上涨或下跌）
✓ 波动率适中
✓ 成交量活跃
✓ 无明显系统性风险

不适合突破策略的市场：
✗ 横盘震荡
✗ 波动率极低
✗ 重大事件前夕
✗ 流动性枯竭
```

## 突破策略风险管理

```
核心原则：
1. 突破失败立即止损
   - 不抱侥幸心理
   - 假突破是常态

2. 控制单笔仓位
   - 突破策略胜率有限
   - 单笔不超过正常仓位70%

3. 分批建仓
   - 首次突破试仓30%
   - 确认后加仓
   - 避免一次性重仓

4. 时间止损
   - 突破后2日内未延续
   - 考虑离场
```
