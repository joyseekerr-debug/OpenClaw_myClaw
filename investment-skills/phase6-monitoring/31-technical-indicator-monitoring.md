# 31. 技术指标监控 (Technical Indicator Monitoring)

## 概念定义

技术指标监控是指对各类技术分析指标进行实时计算和跟踪，在指标达到预设条件或发出信号时触发预警的监控策略。

## 常用监控指标

### 1. 趋势类指标

#### 移动平均线（MA）监控
```
监控类型：
├── 金叉/死叉
│   ├── 短期MA上穿长期MA → 金叉（买入信号）
│   └── 短期MA下穿长期MA → 死叉（卖出信号）
│   └── 常用组合：5/10, 5/20, 10/30, 20/60
│
├── 价格与均线关系
│   ├── 价格上穿MA → 突破信号
│   ├── 价格下穿MA → 跌破信号
│   ├── 价格在MA上方 → 多头趋势
│   └── 价格在MA下方 → 空头趋势
│
└── 均线排列
    ├── 多头排列（短期>中期>长期）→ 强势
    └── 空头排列（短期<中期>长期）→ 弱势

监控实现：
- 实时计算各周期MA
- 比较当前与前一周期MA关系
- 检测交叉事件
```

#### MACD监控
```
监控类型：
├── DIF与DEA交叉
│   ├── 金叉（DIF上穿DEA）→ 买入信号
│   └── 死叉（DIF下穿DEA）→ 卖出信号
│
├── 零轴穿越
│   ├── DIF上穿零轴 → 转强信号
│   └── DIF下穿零轴 → 转弱信号
│
├── MACD柱状图
│   ├── 由负转正 → 动能转强
│   ├── 由正转负 → 动能转弱
│   ├── 柱状图扩大 → 动能增强
│   └── 柱状图缩小 → 动能减弱
│
└── 背离信号
    ├── 顶背离：价格新高，MACD未新高
    └── 底背离：价格新低，MACD未新低
```

### 2. 震荡类指标

#### RSI监控
```
监控类型：
├── 超买超卖
│   ├── RSI > 70 → 超买预警
│   ├── RSI > 80 → 严重超买
│   ├── RSI < 30 → 超卖预警
│   └── RSI < 20 → 严重超卖
│
├── 中轴穿越
│   ├── RSI上穿50 → 转强
│   └── RSI下穿50 → 转弱
│
└── 背离信号
    ├── 顶背离：价格新高，RSI未新高
    └── 底背离：价格新低，RSI未新低
```

#### KDJ监控
```
监控类型：
├── 金叉/死叉
│   ├── K上穿D → 金叉
│   └── K下穿D → 死叉
│
├── 极端区域
│   ├── K、D、J > 80 → 超买区
│   └── K、D、J < 20 → 超卖区
│
└── J线极端值
    ├── J > 100 → 严重超买
    └── J < 0 → 严重超卖
```

### 3. 波动类指标

#### 布林带（BOLL）监控
```
监控类型：
├── 价格与轨道关系
│   ├── 价格上穿上轨 → 超买/突破
│   ├── 价格跌破下轨 → 超卖/跌破
│   ├── 价格从中轨下方向上突破中轨 → 转强
│   └── 价格从中轨上方向下跌破中轨 → 转弱
│
├── 带宽变化
│   ├── 带宽收窄 → 即将变盘
│   └── 带宽扩大 → 趋势加速
│
└── 轨道方向
    ├── 三轨向上 → 上升趋势
    ├── 三轨向下 → 下降趋势
    └── 轨道走平 → 震荡整理
```

## 多指标共振监控

### 共振信号
```
买入共振：
- MA金叉 + MACD金叉 + RSI从超卖回升
- 价格在布林下轨 + KDJ金叉 + 缩量企稳
- 多指标同时发出买入信号

卖出共振：
- MA死叉 + MACD死叉 + RSI从超买回落
- 价格在布林上轨 + KDJ死叉 + 放量滞涨
- 多指标同时发出卖出信号

监控权重：
- 趋势指标（MA、MACD）：40%
- 震荡指标（RSI、KDJ）：30%
- 波动指标（BOLL、ATR）：20%
- 成交量指标：10%
```

## 技术指标监控配置

### 参数设置
```
默认参数（可自定义）：
├── MA：5, 10, 20, 30, 60, 120, 250
├── MACD：12, 26, 9
├── RSI：14
├── KDJ：9, 3, 3
├── BOLL：20, 2
└── ATR：14

个性化参数：
- 短线交易：缩短周期（MA 3/5/8）
- 长线投资：延长周期（MA 20/60/120）
- 高波动标的：调整布林带宽度
```

### 监控频率
```
实时监控（分钟级）：
- 金叉/死叉信号
- 超买超卖突破
- 价格突破轨道

定时监控（日终）：
- 背离信号确认
- 多指标评分
- 趋势状态汇总
```

## 技术指标监控报表

```
日终技术报告：
├── 金叉标的列表
│   └── MA金叉、MACD金叉、KDJ金叉
├── 死叉标的列表
│   └── MA死叉、MACD死叉、KDJ死叉
├── 超买超卖列表
│   └── RSI>70或<30的标的
├── 背离信号
│   └── 顶背离、底背离
└── 综合评分
    └── 多指标共振评分排序
```
