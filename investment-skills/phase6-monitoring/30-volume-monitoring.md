# 30. 成交量监控 (Volume Monitoring)

## 概念定义

成交量监控是指对标的的交易量进行实时跟踪，识别成交量异常变化，并据此判断市场情绪和潜在趋势变化的监控策略。

## 成交量的意义

### 量价关系基础
```
价涨量增：
- 上涨趋势确认
- 资金流入积极
- 多头力量强劲

价涨量缩：
- 上涨动力不足
- 可能是诱多
- 需警惕反转

价跌量增：
- 下跌趋势确认
- 资金出逃明显
- 空头力量强劲

价跌量缩：
- 下跌动力减弱
- 可能接近底部
- 关注企稳信号
```

### 成交量指标
```
1. 量比
   公式：量比 = 现成交总手数 / (过去5日平均每分钟成交量 × 当日累计开市时间)
   意义：
   - 量比>1：放量
   - 量比<1：缩量
   - 量比>2：明显放量
   - 量比>5：异常放量

2. 换手率
   公式：换手率 = 成交量 / 流通股本 × 100%
   意义：
   - 换手率低（<3%）：交投清淡
   - 换手率中（3-7%）：正常活跃
   - 换手率高（>7%）：非常活跃
   - 换手率极高（>20%）：可能变盘

3. 成交额
   意义：反映资金实际参与度
   - 比成交量更能体现大资金动向
   - 大盘股看成交额更准确
```

## 成交量监控策略

### 1. 放量监控
```
异常放量预警：
├── 条件
│   ├── 量比 > 3
│   ├── 换手率 > 10%
│   └── 成交额 > 近期平均2倍
├── 解读
│   ├── 突破放量 → 趋势启动
│   ├── 高位放量 → 可能是顶
│   ├── 低位放量 → 可能是底
│   └── 横盘放量 → 即将变盘
└── 应对
    ├── 确认方向后跟进
    ├── 不明朗时观望
    └── 高位放量考虑减仓
```

### 2. 缩量监控
```
异常缩量预警：
├── 条件
│   ├── 量比 < 0.5
│   ├── 换手率创近期新低
│   └── 成交额萎缩明显
├── 解读
│   ├── 上涨缩量 → 上涨乏力
│   ├── 下跌缩量 → 抛压减轻
│   ├── 盘整缩量 → 等待变盘
│   └── 底部缩量 → 地量地价
└── 应对
    ├── 缩量上涨不追高
    ├── 缩量下跌等企稳
    └── 地量关注反转机会
```

### 3. 量价背离监控
```
顶背离：
- 价格创新高
- 成交量未创新高
- 上涨动能减弱
- 预警：可能见顶

底背离：
- 价格创新低
- 成交量未创新低
- 下跌动能减弱
- 预警：可能见底

监控实现：
- 比较N日价格高点与成交量高点
- 价格新高 but 成交量未新高 → 顶背离预警
- 价格新低 but 成交量未新低 → 底背离预警
```

## 成交量监控参数设置

### 阈值设置
```
大盘股（如茅台）：
- 放量：成交额 > 20亿（或日均2倍）
- 缩量：成交额 < 5亿（或日均0.5倍）

中盘股：
- 放量：量比 > 3 且 换手率 > 8%
- 缩量：量比 < 0.5 且 换手率 < 1%

小盘股：
- 放量：量比 > 5 且 换手率 > 15%
- 缩量：量比 < 0.3 且 换手率 < 2%
```

### 时段设置
```
早盘放量（09:30-10:30）：
- 意义：全天趋势预示
- 放量上涨 → 可能收阳
- 放量下跌 → 可能收阴

午盘放量（13:00-14:00）：
- 意义：趋势加速或反转
- 需结合上午走势判断

尾盘放量（14:30-15:00）：
- 意义：机构调仓或消息刺激
- 关注收盘前动向
```

## 成交量监控实现

### 监控逻辑
```
伪代码：
function check_volume_alert(symbol):
    current_volume = get_current_volume(symbol)
    avg_volume = get_avg_volume(symbol, days=5)
    ratio = current_volume / avg_volume
    
    # 放量监控
    if ratio > volume_spike_threshold:
        if price_change > 0:
            alert_type = "放量上涨"
        else:
            alert_type = "放量下跌"
        send_alert(symbol, alert_type, ratio)
    
    # 缩量监控
    if ratio < volume_drop_threshold:
        alert_type = "异常缩量"
        send_alert(symbol, alert_type, ratio)
    
    # 量价背离监控
    if check_price_volume_divergence(symbol):
        send_alert(symbol, "量价背离")
```
