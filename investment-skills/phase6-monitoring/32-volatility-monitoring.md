# 32. 波动率监控 (Volatility Monitoring)

## 概念定义

波动率监控是指对市场或标的的价格波动程度进行实时跟踪，识别波动率异常变化，并据此调整交易策略和风险管理的监控机制。

## 波动率指标

### 1. 历史波动率（Historical Volatility）
```
计算公式：
HV = σ × √T

其中：
σ = 收益率标准差（年化）
T = 时间周期（通常252个交易日）

计算步骤：
1. 计算日收益率：r_t = ln(P_t / P_{t-1})
2. 计算收益率标准差：σ = std(r)
3. 年化：HV = σ × √252

监控意义：
- 反映过去一段时间价格波动程度
- 用于预测未来波动范围
- 期权定价的重要输入
```

### 2. ATR（平均真实波幅）
```
计算公式：
TR = max(High-Low, |High-Close_{t-1}|, |Low-Close_{t-1}|)
ATR = SMA(TR, N) 通常N=14

监控意义：
- 反映日内波动幅度
- 用于动态止损设置
- 判断当前市场活跃度

阈值参考：
- ATR率（ATR/价格）< 1%：低波动
- ATR率 1-3%：中等波动
- ATR率 > 3%：高波动
```

### 3. VIX（波动率指数）
```
概念：
- 标普500指数期权的隐含波动率
- 被称为"恐慌指数"
- 反映市场对未来30天波动的预期

监控意义：
├── VIX < 15：市场平静，风险较低
├── VIX 15-25：正常波动
├── VIX 25-35：市场焦虑，风险升高
├── VIX 35-50：市场恐慌
└── VIX > 50：极度恐慌，危机级别

A股对应：
- 中国波指（iVIX）- 已暂停发布
- 50ETF期权隐含波动率
- 300ETF期权隐含波动率
```

## 波动率监控策略

### 1. 波动率趋势监控
```
波动率上升：
├── 特征
│   ├── ATR持续扩大
│   ├── 价格区间扩大
│   └── 日内波动加剧
├── 应对
│   ├── 降低仓位
│   ├── 放宽止损
│   ├── 减少交易频率
│   └── 考虑买入期权保护
└── 机会
    └── 趋势策略表现更好

波动率下降：
├── 特征
│   ├── ATR持续缩小
│   ├── 价格区间收窄
│   └── 日内波动减小
├── 应对
│   ├── 增加仓位
│   ├── 收紧止损
│   ├── 准备 breakout 交易
│   └── 考虑卖出期权收权利金
└── 机会
    └── 均值回归策略表现更好
```

### 2. 波动率突破监控
```
波动率突破：
- ATR突破近期区间上轨 → 波动率扩大预警
- ATR跌破近期区间下轨 → 波动率收缩预警

实现：
1. 计算ATR的布林带（20日，2倍标准差）
2. ATR上穿上轨 → 高波动预警
3. ATR下穿下轨 → 低波动预警
```

### 3. GARCH类模型监控
```
概念：
- GARCH模型预测条件波动率
- 考虑波动率聚类特征
- 高波动后倾向于继续高波动

应用：
- 预测未来波动率
- 动态调整仓位
- 期权定价调整

简化监控：
- 今日波动率 vs 预测波动率
- 实际 > 预测 → 异常波动
```

## 波动率与交易策略

### 策略适配
```
高波动环境（HV > 30%）：
├── 适合策略
│   ├── 趋势跟踪
│   ├── 突破策略
│   └── 动量策略
├── 不适合策略
│   ├── 均值回归
│   ├── 卖方期权
│   └── 高杠杆策略
└── 风险管理
    ├── 降低仓位至正常50%
    ├── 扩大止损至2倍ATR
    └── 缩短持仓周期

低波动环境（HV < 15%）：
├── 适合策略
│   ├── 均值回归
│   ├── 卖方期权
│   ├── 套利策略
│   └── 高杠杆策略
├── 不适合策略
│   ├── 趋势跟踪（假突破多）
│   └── 动量策略
└── 风险管理
    ├── 增加仓位
    ├── 收紧止损
    └── 准备 breakout
```

### 波动率交易（Volatility Trading）
```
做多波动率：
- 买入期权（跨式、宽跨式）
- 买入VIX期货/ETF
- 预期波动率上升时获利

做空波动率：
- 卖出期权
- 卖出VIX期货/ETF
- 预期波动率下降时获利

风险：
- 做空波动率风险极高
- 波动率可能突然飙升
- 严格风控必不可少
```

## 波动率监控配置

### 监控参数
```
监控周期：
- 短期波动率：5-10日
- 中期波动率：20-60日
- 长期波动率：120-250日

预警阈值：
- 波动率突破20日新高 → 高波动预警
- 波动率跌破20日新低 → 低波动预警
- 波动率单日变化 > 20% → 异常预警
```

### 监控报表
```
日终波动率报告：
├── 各标的波动率排名
│   └── 高波动、低波动标的列表
├── 波动率变化
│   └── 波动率上升/下降最大标的
├── 波动率分位
│   └── 当前波动率在历史中的位置
└── 波动率预警
    └── 突破、异常变化提醒
```
