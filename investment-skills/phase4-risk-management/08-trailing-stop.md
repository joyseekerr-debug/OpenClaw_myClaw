# 8. 移动止损 (Trailing Stop)

## 概念定义

移动止损（Trailing Stop），又称跟踪止损、追踪止损，是一种动态调整止损价位的策略。随着价格向有利方向移动，止损位也相应上移（多头）或下移（空头），从而锁定部分利润，同时保留参与更大行情的机会。

## 核心原理

### 运作机制
```
多头移动止损示例：
入场价：100元
初始止损：95元（-5%）

价格上涨至110元 → 止损上移至105元（锁定5元利润）
价格上涨至120元 → 止损上移至114元（锁定14元利润）
价格回落至114元 → 触发止损，盈利14%
```

### 核心思想
- **保护利润**：确保不会从最高点大幅回撤
- **让利润奔跑**：不限制上行空间
- **机械执行**：避免情绪干扰

## 移动止损类型

### 1. 固定比例移动止损
- **公式**：止损价 = 最高价 × (1 - 回撤比例)
- **常用比例**：5%、8%、10%、15%
- **优点**：简单直观
- **缺点**：未考虑波动率差异

### 2. 固定金额移动止损
- **公式**：止损价 = 最高价 - 固定金额
- **适用**：价格区间稳定的品种
- **示例**：最高50元，固定回撤2元，止损48元

### 3. ATR移动止损
- **公式**：止损价 = 最高价 - N × ATR
- **N值选择**：通常2-3倍ATR
- **优点**：自适应市场波动
- **缺点**：震荡期可能过于敏感

### 4. 均线移动止损
- **短期均线**：跌破5日/10日均线
- **中期均线**：跌破20日/60日均线
- **长期均线**：跌破120日/250日均线
- **选择依据**：交易周期匹配

### 5. 抛物线SAR
- **原理**：SAR点随趋势自动上移
- **优点**：自带加速因子
- **缺点**：震荡市频繁信号

### 6. 布林带移动止损
- **中轨止损**：跌破布林带中轨
- **下轨止损**：触及布林带下轨
- **适用**：趋势+波动结合

## 参数选择指南

### 回撤比例选择
| 交易风格 | 回撤比例 | 适用场景 |
|----------|----------|----------|
| 超短线 | 2%-3% | 日内交易 |
| 短线 | 5%-8% | 数日至数周 |
| 中线 | 10%-15% | 数周至数月 |
| 长线 | 15%-25% | 数月至数年 |

### ATR倍数选择
```
保守（低波动）：1.5-2倍ATR
中性（一般）：2-3倍ATR
激进（高波动）：3-5倍ATR
```

## 移动止损触发条件

### 价格触发
- 收盘价跌破移动止损位
- 最低价触及移动止损位
- 盘中跌破 vs 收盘确认

### 时间触发
- 每日收盘后调整
- 固定时间间隔检查
- 新K线形成时

## 移动止损策略组合

### 初级组合
- 固定比例10%移动止损
- 硬条件单执行

### 中级组合
- 2倍ATR移动止损
- 结合时间确认（如连续2日跌破）

### 高级组合
```
第1阶段：3倍ATR移动止损（趋势初期，容忍波动）
第2阶段：2倍ATR移动止损（趋势确立，收紧保护）
第3阶段：1.5倍ATR移动止损（趋势末期，严格保护）

或：
短期仓位：5%固定比例
中期仓位：10日均线
长期仓位：20日均线
```

## 移动止损的优缺点

### 优点 ✅
- 自动锁定利润
- 无需盯盘决策
- 克服贪婪心理
- 保留上行空间
- 机械执行，纪律性强

### 缺点 ❌
- 震荡市频繁触发
- 无法卖在最高点
- 参数选择困难
- 跳空可能大幅滑点
- 单独使用可能错失大行情

## 移动止损最佳实践

### 入场即设
- 开仓同时设置移动止损
- 避免利润回吐后被动决策

### 结合分级止盈
```
盈利50% → 移动止损至成本价（保本）
盈利100% → 移动止损至盈利50%位置
盈利200% → 移动止损至盈利100%位置
```

### 重要价位保护
- 移动止损不跌破关键支撑位
- 整数关口、均线、趋势线
- 给予正常回调空间

### 波动率调整
- 高波动期放宽移动止损
- 低波动期收紧移动止损
- 公告/事件前考虑收紧

## 移动止损计算公式速查

```
固定比例：
止损价 = 最高价 × (1 - 回撤比例)

固定金额：
止损价 = 最高价 - 固定回撤额

ATR止损：
止损价 = 最高价 - N × ATR(14)

均线止损：
止损价 = MA(N) 的上一周期值
```
