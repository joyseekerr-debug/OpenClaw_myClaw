# 10. 组合分散化 (Portfolio Diversification)

## 概念定义

组合分散化是指通过投资于多种不同特征（如行业、地区、资产类别）的标的，降低投资组合整体风险的策略。核心原理是利用资产间的低相关性，在不牺牲预期收益的前提下降低波动。

## 核心原理

### 现代投资组合理论（MPT）
哈里·马科维茨（Harry Markowitz）1952年提出：
```
组合方差公式：
σ²p = ΣΣ wi × wj × σi × σj × ρij

关键变量：
ρij = 资产i,j的相关系数（-1到+1）
wi, wj = 资产权重
σi, σj = 资产标准差
```

**关键洞察**：当ρ<1时，组合风险<各资产风险的加权平均

### 分散化效应
```
完全正相关(ρ=1)：无分散效果
完全负相关(ρ=-1)：可完全对冲
实际市场(0<ρ<1)：部分分散效果
```

## 分散化维度

### 1. 资产类别分散
| 资产类别 | 风险等级 | 预期收益 | 相关性 |
|----------|----------|----------|--------|
| 股票 | 高 | 高 | - |
| 债券 | 中 | 中低 | 低 |
| 商品 | 高 | 高 | 中低 |
| 房地产 | 中 | 中 | 中 |
| 现金 | 低 | 低 | 无 |

### 2. 地域分散
- 发达国家 vs 新兴市场
- 不同货币政策区域
- 汇率风险对冲

### 3. 行业分散
- 周期性 vs 防御性
- 成长型 vs 价值型
- 传统 vs 新兴行业

### 4. 因子分散
- 市值因子（大/小盘）
- 价值因子（高/低PB）
- 动量因子（高/低动量）
- 质量因子（高/低ROE）

## 有效分散vs伪分散

### 有效分散 ✅
- 低相关性资产（ρ<0.5）
- 不同经济周期表现
- 真正独立的收益来源

### 伪分散 ❌
- 同行业不同股票（ρ>0.8）
- 同一主题ETF重叠
- 形式分散实质集中

## 分散化极限

### 风险分解
```
总风险 = 系统性风险(市场) + 非系统性风险(个股)
       ↓                  ↓
     不可分散          可分散

分散化只能消除非系统性风险
持有15-30只低相关股票可消除90%非系统性风险
```

### 过度分散
- 管理困难，研究深度不足
- 收益趋于平庸（接近指数）
- 交易成本累积

## 分散化策略

### 核心-卫星策略
```
核心(60-80%)：宽基指数ETF - 稳定收益
卫星(20-40%)：主动选股 - 超额收益
```

### 风险平价策略
- 各资产风险贡献相等
- 低波动资产配高权重
- 高波动资产配低权重

### 全天候策略
- 经济增长/衰退 × 通胀上升/下降
- 四种情景各配25%风险
- 桥水基金著名策略

## 相关性动态变化

⚠️ **危机中的相关性趋同**
- 正常时期：股票间ρ≈0.3-0.5
- 危机时期：股票间ρ→0.8-1.0
- 分散化效果在最需要时下降

**应对**：
- 加入真正的避险资产（国债、黄金、现金）
- 使用期权保护
- 动态调整仓位
