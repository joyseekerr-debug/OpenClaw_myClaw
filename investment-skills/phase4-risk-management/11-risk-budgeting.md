# 11. 风险预算 (Risk Budgeting)

## 概念定义

风险预算（Risk Budgeting）是一种资产配置方法，基于风险而非资金来分配投资组合。它将总风险分解为各资产/策略的风险贡献，确保风险分配符合投资目标和约束。

## 核心概念

### 风险贡献度
```
资产i的风险贡献 = wi × (协方差矩阵 × 权重向量)i / 组合波动率
                = wi × βi × σp

其中：
wi = 资产i的权重
βi = 资产i相对于组合的beta
σp = 组合波动率
```

### 风险预算vs资金预算
| 维度 | 资金预算 | 风险预算 |
|------|----------|----------|
| 分配基础 | 资金金额 | 风险贡献 |
| 相同权重风险 | 不同 | 相同 |
| 低波动资产 | 低配 | 可高配 |
| 高波动资产 | 高配 | 需低配 |

## 风险平价策略

### 原理
使每个资产对组合总风险的贡献相等
```
目标：RC1 = RC2 = ... = RCn = 总风险/n

其中RCi = wi × (Σw × cov)i / σp
```

### 计算示例
假设两资产：
- 债券：波动率5%
- 股票：波动率20%
- 相关系数：0.2

风险平价权重：
- 债券：约80%
- 股票：约20%

这样两资产对组合风险贡献各占50%

## 风险预算应用

### 资产配置层面
```
总风险预算：10%波动率
├── 股票：贡献5%（配置约30%资金）
├── 债券：贡献3%（配置约50%资金）
├── 商品：贡献1.5%（配置约10%资金）
└── 现金：贡献0.5%（配置约10%资金）
```

### 策略配置层面
```
量化策略风险预算：
├── 趋势跟踪：40%风险预算
├── 均值回归：30%风险预算
├── 套利策略：20%风险预算
└── 事件驱动：10%风险预算
```

### 交易员层面
```
基金经理给交易员的日风险预算：
├── 交易员A（资深）：VAR $50,000
├── 交易员B（中级）：VAR $30,000
└── 交易员C（初级）：VAR $10,000
```

## 风险预算实施步骤

### 步骤1：确定总风险容忍度
- 最大可接受波动率
- 最大回撤限制
- VaR限额

### 步骤2：分解风险预算
- 按资产类别分配
- 按策略类型分配
- 按地理区域分配

### 步骤3：计算最优权重
```
使用优化算法（如二次规划）：
最小化：Σ(实际风险贡献 - 目标风险贡献)²
约束：Σwi = 1, wi ≥ 0
```

### 步骤4：监控与再平衡
- 定期监控实际风险贡献
- 偏离目标时调整权重
- 考虑交易成本

## 风险预算的优势

✅ **更科学的风险管理**
- 风险透明可见
- 避免"隐形集中"

✅ **更好的危机应对**
- 高波动资产自动减仓
- 低波动资产提供稳定器

✅ **提高夏普比率**
- 风险分配更高效
- 单位风险收益更高

## 注意事项

⚠️ **局限性**
- 依赖历史相关性（危机中失效）
- 低波动资产可能高配导致收益平庸
- 杠杆使用增加复杂性

⚠️ **实施难点**
- 需要准确的协方差估计
- 频繁再平衡产生成本
- 杠杆比率管理
