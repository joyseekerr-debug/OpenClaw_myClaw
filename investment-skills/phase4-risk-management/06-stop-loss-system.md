# 6. 止损策略体系 (Stop Loss Strategy System)

## 概念定义

止损（Stop Loss）是指在投资亏损达到预定金额或比例时，主动平仓离场以限制损失的交易纪律。止损策略体系是一套完整的规则，包括止损类型选择、止损位设置、止损执行和止损后处理。

## 为什么必须止损

### 数学现实
```
亏损比例 → 回本所需盈利
  -10%  →  +11.1%
  -20%  →  +25.0%
  -30%  →  +42.9%
  -40%  →  +66.7%
  -50%  →  +100%
  -60%  →  +150%
  -70%  →  +233%
  -80%  →  +400%
  -90%  →  +900%
```

**结论**：亏损越深，回本难度呈指数级增长。

## 止损类型

### 1. 固定金额止损
- **定义**：每笔交易最大亏损固定金额
- **公式**：止损价 = 入场价 - (最大亏损金额 / 股数)
- **优点**：绝对风险可控
- **缺点**：未考虑价格波动性

### 2. 固定比例止损
- **定义**：按账户资金百分比止损
- **公式**：最大亏损 = 总资金 × 风险比例（通常1%-3%）
- **优点**：与账户规模挂钩
- **缺点**：仓位大时止损价可能过近

### 3. 技术位止损
- **支撑位/阻力位**：跌破关键价位止损
- **趋势线**：跌破趋势线止损
- **移动平均线**：跌破均线止损
- **形态止损**：形态破坏时止损

### 4. 波动率止损（ATR止损）
- **公式**：止损价 = 入场价 - N × ATR
- **ATR**：平均真实波幅（通常14日）
- **N**：倍数（通常2-3倍）
- **优点**：自适应市场波动

### 5. 时间止损
- **定义**：持仓超过预定时间未盈利即离场
- **适用**：短线交易、事件驱动
- **原理**：机会成本管理

## 止损位设置方法

### 多头止损设置
```
1. 支撑位下方：止损 = 支撑位 × 0.98
2. 趋势线下方：止损 = 趋势线价格 - 缓冲
3. ATR止损：止损 = 入场价 - 2×ATR
4. 固定比例：止损 = 入场价 × (1 - 风险比例)
5. 时间止损：持有N天未盈利即离场
```

### 止损缓冲原则
- 避免整数关口（如10.00元设9.98元）
- 考虑日内波动噪音
- 给趋势正常回调空间

## 止损执行纪律

###  hard stop vs mental stop
| 类型 | 特点 | 推荐度 |
|------|------|--------|
| 硬止损（条件单） | 系统自动执行 | ⭐⭐⭐⭐⭐ |
| 软止损（心理） | 手动执行 | ⭐⭐ |

### 止损后禁止行为
❌ 立即反向开仓
❌ 取消止损重新入场
❌ 赌气加仓摊平

### 止损后正确做法
✅ 记录止损原因
✅ 分析是否有改进空间
✅ 等待下一个符合规则的机会

## 常见止损错误

### 1. 不止损
- 原因：侥幸心理、不愿承认错误
- 结果：小亏变大亏

### 2. 止损过窄
- 原因：恐惧亏损
- 结果：频繁被震出（止损-追涨-再止损）

### 3. 止损过宽
- 原因：怕踏空
- 结果：单次亏损过大

### 4. 移动止损过慢
- 原因：贪婪
- 结果：盈利变亏损

### 5. 情绪止损
- 原因：受情绪影响临时决定
- 结果：无纪律，长期必亏

## 止损策略组合

### 初级策略
- 固定比例止损（每笔2%）
- 硬止损条件单

### 中级策略
- ATR动态止损
- 技术位 + 时间双止损

### 高级策略
- 分级止损（部分仓位不同止损位）
- 波动率自适应止损
- 多时间框架止损确认

## 止损心理建设

### 正确认知
- 止损是交易成本，不是错误
- 接受"这笔交易错了"的事实
- 止损保住的是本金和机会

### 执行技巧
- 设置条件单，避免临场犹豫
- 止损后离开屏幕，冷静后再分析
- 将止损视为"保险费"

## 止损策略检查清单

```
□ 每笔交易前明确止损位
□ 止损位基于技术分析或波动率
□ 单笔风险不超过账户2%
□ 使用条件单自动执行
□ 止损后记录并分析
□ 定期回顾止损有效性
```
