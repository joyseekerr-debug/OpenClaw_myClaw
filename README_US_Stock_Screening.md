## 📊 关键指标说明

### 筛选指标
- **相对超额**: 个股3M涨幅 - ETF基准3M涨幅
- **Z-Score**: (相对超额 - 均值) / 标准差
- **阈值**: Z-Score < -1.0 为负超额超过1个标准差

### 基本面指标
- **P/E TTM**: 滚动12个月市盈率
- **EV/EBITDA**: 企业价值倍数
- **分析师覆盖**: 机构研究覆盖数量

### 诊断评分 (0-8分)
- **情绪面**: 负面舆情占比与趋势 (0-2分)
- **信息面**: 分析师覆盖密度 (0-2分)
- **流动性**: 流通市值规模 (0-2分)
- **筹码面**: 机构持仓变化 (0-2分)

**解读**:
- 4-6分: 暂时性因素 → 推荐配置
- 0-3分: 结构性忽视 → 谨慎配置
