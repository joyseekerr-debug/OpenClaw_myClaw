# 价格行为学 (Price Action) 完整学习指南

> 基于价格本身的交易艺术——不依赖技术指标，读懂市场语言

---

## 第一部分：基础概念

### 1.1 什么是价格行为学？

**定义**：价格行为学是一种通过分析历史价格走势来预测未来价格变动的技术分析方法。它不依赖任何技术指标，只关注价格本身。

**核心理念**：
- 价格反映一切信息
- 历史会重演
- 价格沿趋势运动
- 市场行为包容消化一切

**与指标交易的区别**：

| 价格行为 | 指标交易 |
|---------|---------|
| 直接分析价格 | 分析价格的衍生品 |
| 反应迅速 | 有滞后性 |
| 主观性强 | 信号明确但延迟 |
| 需要更多经验 | 易于量化回测 |

---

### 1.2 K线基础

#### K线四要素
- **开盘价**：该周期第一笔成交价
- **收盘价**：该周期最后一笔成交价
- **最高价**：该周期最高成交价
- **最低价**：该周期最低成交价

#### K线分类

**按实体大小**：
- 大阳线/大阴线：实体占K线70%以上，趋势强劲
- 中阳线/中阴线：实体适中，趋势正常
- 小阳线/小阴线：实体较小，趋势犹豫
- 十字星：开盘价≈收盘价，多空平衡

**按影线长度**：
- 长腿十字星：上下影线长，反转信号
- 锤子线：下影线长，底部信号
- 倒锤子：上影线长，顶部信号
- 光头光脚：无影线，趋势极强

## 第二部分：核心形态

### 2.1 单根K线形态

#### 锤子线 (Hammer) / 上吊线 (Hanging Man)
```
    形态：小实体 + 长下影线（实体2倍以上）
    位置：锤子线在下跌底部，上吊线在上涨顶部
    含义：市场拒绝更低价格
    
    锤子线:          上吊线:
      │                │
      │                │
    ━━┷━━           ━━┯━━
      │                │
      │                │
```

**确认条件**：
1. 出现在趋势末端
2. 下影线长度≥实体2倍
3. 上影线极短或没有
4. 下一根K线确认方向

#### 流星线 (Shooting Star)
```
    形态：小实体 + 长上影线
    位置：上涨顶部
    含义：市场拒绝更高价格
    
      │
      │
    ━━┯━━
      │
      │
```

#### 吞没形态 (Engulfing)

**看涨吞没**：
- 第一根：阴线
- 第二根：阳线，完全包含第一根
- 位置：下跌底部

**看跌吞没**：
- 第一根：阳线
- 第二根：阴线，完全包含第一根
- 位置：上涨顶部

```
看涨吞没:         看跌吞没:
  ┃│              │┃
  ┃│              │┃
━━┿━━           ━━┿━━
  │┃              ┃│
  │┃              ┃│
```

#### 刺透形态 (Piercing Pattern) / 乌云盖顶 (Dark Cloud Cover)

**刺透形态**（看涨）：
- 第一根：大阴线
- 第二根：阳线开盘价低于前低，收盘价深入前实体50%以上

**乌云盖顶**（看跌）：
- 第一根：大阳线
- 第二根：阴线开盘价高于前高，收盘价深入前实体50%以上

### 2.2 多根K线组合

#### 晨星 (Morning Star) / 暮星 (Evening Star)

**晨星**（底部反转）：
1. 第一根：大阴线
2. 第二根：小实体（十字星最佳），跳空低开
3. 第三根：大阳线，收盘价深入第一根实体

**暮星**（顶部反转）：
1. 第一根：大阳线
2. 第二根：小实体，跳空高开
3. 第三根：大阴线，收盘价深入第一根实体

```
晨星:              暮星:
  │                 ┃
  │                 ┃
━━┿━━            ━━┿━━
  │┃                │
  │┃   ┃            │   ┃
━━┿━━━┿━━        ━━┿━━━┿━━
      ┃│                ┃│
      ┃│                ┃│
```

#### 三只乌鸦 (Three Black Crows) / 三白兵 (Three White Soldiers)

**三白兵**（强烈看涨）：
- 连续三根阳线
- 每根开盘价在前一根实体内
- 每根收盘价接近高点

**三只乌鸦**（强烈看跌）：
- 连续三根阴线
- 每根开盘价在前一根实体内
- 每根收盘价接近低点

## 第三部分：支撑与阻力

### 3.1 核心概念

**支撑位**：价格下跌时遇到买盘力量，难以跌破的水平
**阻力位**：价格上涨时遇到卖盘力量，难以突破的水平

**形成原因**：
1. 记忆效应（前期高低点）
2. 成本集中区（大量成交区）
3. 心理关口（整数位、历史重要价位）
4. 移动平均线

### 3.2 支撑阻力的识别

#### 水平支撑阻力
```
价格:          
    ──────── 阻力R2
        │
    ──────── 阻力R1（前高）
      ╱ ╲
     ╱   ╲
────┘     └──── 支撑S1（前低）
        │
    ──────── 支撑S2
```

**识别方法**：
1. 至少两次触及同一水平
2. 价格在该水平明显反转
3. 成交量在该水平放大

#### 动态支撑阻力

**趋势线**：
- 上升趋势线：连接两个以上低点，斜向上
- 下降趋势线：连接两个以上高点，斜向下

**趋势线画法规则**：
1. 至少两点确定一条趋势线
2. 第三点确认有效性
3. 触点越多，趋势线越有效
4. 陡峭趋势线易被突破

```
上升趋势线:          下降趋势线:
    ╱               ╲
   ╱ ╲               ╲
  ╱   ╲               ╲
 ╱     ╲───            ╲───
╱                       ╲
```

### 3.3 支撑阻力的转换

**核心原则**：支撑被跌破后变阻力，阻力被突破后变支撑

```
突破前:              突破后:
    │                   │
────┿──── 阻力      ────┿──── 支撑
    │                   │
    │                   │
    │    价格           │    价格
    │                   │
────┘                   └────
```

## 第四部分：图表形态

### 4.1 反转形态

#### 头肩顶/底 (Head & Shoulders)

**头肩顶**（看跌）：
```
       ┌─┐ 左肩
     ┌─┘ └─┐
    ─┘     └────┐
                ┌─┐ 头（最高）
              ┌─┘ └─┐
    ─────────┘     └────┐
                      ┌─┐ 右肩
                    ┌─┘ └─┐
    ───────────────┘       └────
                      ↑ 颈线
```

**交易规则**：
- 入场：跌破颈线
- 止损：右肩高点上方
- 目标：头部到颈线的垂直距离

#### 双顶/双底 (Double Top/Bottom)

**双顶**（M头）：
```
    ┌─┐       ┌─┐
    │ │       │ │
────┘ └───────┘ └──────
        └─────┘
          颈线
```

**双底**（W底）：
```
────┐       ┌───────┐
    │       │       │
    └───────┘       └─┘
        ┌─────┐
        颈线
```

#### 圆弧顶/底 (Rounding Top/Bottom)
- 缓慢形成的顶部/底部
- 成交量逐步萎缩后放大
- 反转力度强，持续时间长

### 4.2 持续形态

#### 三角形

**对称三角形**：
- 上轨下降，下轨上升
- 多空力量趋于平衡
- 突破方向与之前趋势一致概率大

```
    ╲
     ╲  /
      ╲/
      /\
     /  ╲
    /
```

**上升三角形**：
- 水平上轨（阻力），上升下轨（支撑）
- 看涨形态，向上突破概率大

**下降三角形**：
- 水平下轨（支撑），下降上轨（阻力）
- 看跌形态，向下突破概率大

#### 旗形与三角旗

**旗形**（Flag）：
- 与主趋势相反的小通道
- 倾斜方向与主趋势相反
- 持续时间较短（1-3周）

**三角旗**（Pennant）：
- 对称三角形的小型版本
- 出现在快速上涨/下跌后

```
主趋势向上 + 旗形:
    │
    │    ╱╲
    │   ╱  ╲
    │  ╱    ╲
    │ ╱      ╲
    │╱        ╲___
    │
```

#### 矩形（箱体）
- 价格在水平支撑阻力间震荡
- 突破后目标为箱体高度
- 可作为积累或派发阶段

## 第五部分：供需区域

### 5.1 供需理论基础

**需求区**（Demand Zone）：
- 大量买单聚集区域
- 价格下跌时买方积极入场
- 表现为强势上涨起点

**供给区**（Supply Zone）：
- 大量卖单聚集区域
- 价格上涨时卖方积极出场
- 表现为强势下跌起点

### 5.2 识别供需区

#### 需求区特征：
1. 强势阳线起点
2. 多根K线实体重叠区
3. 成交量放大
4. 价格快速离开该区域

#### 供给区特征：
1. 强势阴线起点
2. 多根K线实体重叠区
3. 成交量放大
4. 价格快速离开该区域

```
供给区:            需求区:
  ╱╲                 ╲
 ╱  ╲                 ╲╱
╱    ╲___               ╲___
                              ╲
                               ╲
```

### 5.3 供需区交易策略

**买入设置**：
1. 识别需求区
2. 等待价格回调至需求区
3. 在需求区内出现看涨K线形态
4. 止损设在需求区下方

**卖出设置**：
1. 识别供给区
2. 等待价格反弹至供给区
3. 在供给区内出现看跌K线形态
4. 止损设在供给区上方

## 第六部分：多时间框架分析

### 6.1 时间框架层次

**推荐框架组合**：
- 大趋势：周线/日线
- 中趋势：日线/4小时
- 交易框架：1小时/15分钟
- 入场框架：15分钟/5分钟

**自上而下分析流程**：
1. 月线/周线确定主要趋势方向
2. 日线寻找关键支撑阻力
3. 4小时/1小时寻找交易机会
4. 15分钟/5分钟精确入场

### 6.2 多时间框架确认

**顺大势交易原则**：
- 周线趋势向上 → 只做多
- 日线回调至支撑 → 寻找买入机会
- 小时线出现反转形态 → 入场

```
周线: ↑↑↑ (上升趋势)
日线: ↓  (回调)
小时: ━━┷━━ (锤子线，入场)
```

## 第七部分：交易策略体系

### 7.1 趋势跟踪策略

**策略逻辑**：顺势而为，让利润奔跑

**入场条件**：
1. 日线趋势明确（20EMA之上/下）
2. 价格回调至关键支撑/阻力
3. 出现确认K线形态
4. 成交量配合

**止损设置**：
- 保守：信号K线最低点/最高点
- 中等：前一根K线极值
- 宽松：关键支撑/阻力位

**止盈方法**：
- 固定盈亏比（1:2, 1:3）
- 移动止损（追踪趋势）
- 目标位（前高/前低，形态目标）

### 7.2 区间交易策略

**适用市场**：震荡行情，无明显趋势

**操作方法**：
1. 确定区间边界（支撑/阻力）
2. 价格触及支撑且出现看涨信号 → 买入
3. 价格触及阻力且出现看跌信号 → 卖出
4. 区间中央位置不做单

**风险控制**：
- 严格止损于区间外侧
- 区间收窄时减仓或退出
- 突破区间立即止损

### 7.3 突破交易策略

**真突破vs假突破**：

| 真突破 | 假突破 |
|--------|--------|
| 成交量放大 | 成交量萎缩 |
| 收盘价突破 | 仅影线突破 |
| 突破后不回撤 | 快速回到区间内 |
| 多时间框架确认 | 单一时间框架 |

**突破交易策略**：
1. 等待价格突破关键水平
2. 确认突破有效性（收盘、量能）
3. 回撤至突破位附近入场
4. 止损设在假突破极值点

## 第八部分：风险管理

### 8.1 仓位管理原则

**固定比例法**：
- 单笔风险 ≤ 账户总额2%
- 每笔交易计算头寸规模
- 公式：头寸 = (账户 × 风险%) ÷ 止损点数

**金字塔加仓法**：
- 第一笔：试探性仓位
- 第二笔：趋势确认后加仓
- 第三笔：趋势加速后加仓
- 每加仓一次，止损上移

### 8.2 止损策略

**技术止损**：
- 设在支撑/阻力外侧
- 设在形态极值点
- 设在趋势线外侧

**时间止损**：
- 入场后N根K线未按预期运动 → 平仓
- 适用于时间敏感型交易

**资金止损**：
- 日/周最大亏损限额
- 触及限额停止交易

### 8.3 情绪管理

**交易心理戒律**：
1. 不要让盈利变成亏损
2. 不要试图抄底摸顶
3. 不要 revenge trading（报复性交易）
4. 不要 over trading（过度交易）
5. 不要移动止损扩大亏损

**交易日志记录**：
- 入场理由
- 入场价格、止损、目标
- 实际出场及原因
- 情绪状态
- 复盘总结

## 第九部分：进阶技巧

### 9.1 价格行为与成交量

**量价关系**：
- 价格上涨 + 成交量放大 → 趋势健康
- 价格上涨 + 成交量萎缩 → 趋势可能衰竭
- 价格突破 + 成交量放大 → 真突破概率高
- 价格突破 + 成交量萎缩 → 假突破概率高

### 9.2 假突破交易

**假突破特征**：
1. 快速突破关键水平
2. 成交量不配合
3. 迅速回到区间内
4. 形成长影线

**交易策略**：
- 等待假突破确认
- 反向入场
- 止损设在突破极值点
- 目标为区间另一侧

### 9.3 回调与反转的区分

| 回调特征 | 反转特征 |
|---------|---------|
| 与主趋势相反的小波段 | 改变原有趋势结构 |
| 成交量缩小 | 成交量放大 |
| 形态简单（ABC或简单K线） | 形态复杂（头肩、双顶等） |
| 持续时间短 | 持续时间长 |
| 不突破关键支撑/阻力 | 突破关键支撑/阻力 |

## 第十部分：实战练习

### 练习1：K线形态识别
在图表中寻找以下形态各10个：
- 锤子线/上吊线
- 吞没形态
- 晨星/暮星
- 三只乌鸦/三白兵

### 练习2：支撑阻力绘制
在历史图表上：
1. 标记所有重要水平位
2. 绘制趋势线
3. 观察支撑阻力的转换
4. 统计突破/反转概率

### 练习3：模拟交易
制定交易计划：
1. 确定交易框架（日线/4小时/1小时）
2. 建立入场、止损、止盈规则
3. 模拟交易50笔
4. 统计胜率、盈亏比、期望值

### 练习4：多时间框架分析
选择单一品种：
1. 周线判断大趋势
2. 日线找关键位
3. 4小时等待回调
4. 1小时寻找入场
5. 记录每次分析的准确性

### 关键要点总结

**价格行为交易核心**：
1. 趋势是你的朋友
2. 支撑阻力是关键
3. K线形态确认信号
4. 风险管理决定生存
5. 纪律执行决定盈利

**成功公式**：
正期望值 = (胜率 × 平均盈利) - (败率 × 平均亏损)

> 价格行为学不是圣杯，而是一种概率优势。长期坚持正向期望值的策略，时间会成为你的朋友。
