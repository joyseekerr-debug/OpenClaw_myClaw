# 支撑阻力位（Support & Resistance）

## 一、概念定义

支撑和阻力是技术分析中最基础、最重要的概念，代表市场价格运动中的关键"关卡"。

**支撑位（Support）：**
- 价格下跌时遇到的"地板"，能够阻止价格继续下跌
- 代表买方力量>卖方力量的价格区域
- 多次测试未跌破的价位

**阻力位（Resistance）：**
- 价格上涨时遇到的"天花板"，能够阻止价格继续上涨
- 代表卖方力量>买方力量的价格区域
- 多次测试未突破的价位

**核心思想：**
- 历史会重演：曾经的重要价位会被市场记住
- 心理关口：整数位、历史高低点具有特殊意义
- 角色互换：支撑位跌破后变成阻力位，阻力位突破后变成支撑位

**形成原因：**
1. **心理因素：** 历史高低点、整数关口
2. **筹码分布：** 大量成交密集区
3. **机构成本：** 主力建仓/出货成本区
4. **技术形态：** 颈线、趋势线

## 二、识别方法（文字描述）

### 水平支撑阻力

**识别方法：**
1. 找出价格多次触及但未突破的水平价位
2. 至少2-3次测试确认
3. 测试次数越多，支撑/阻力越强

**常见水平位：**
- 历史高点和低点
- 前期密集成交区
- 整数关口（如100元、50港元）
- 涨跌停价格

### 趋势线支撑阻力

**上升趋势线（支撑）：**
- 连接两个以上依次抬高的低点
- 价格回落到趋势线时获得支撑
- 趋势线被跌破，上升趋势可能结束

**下降趋势线（阻力）：**
- 连接两个以上依次降低的高点
- 价格反弹到趋势线时遇到阻力
- 趋势线被突破，下降趋势可能结束

### 均线支撑阻力

- 短期均线（5日、10日）：动态支撑/阻力
- 中期均线（20日、60日）：较强支撑/阻力
- 长期均线（120日、250日）：最强支撑/阻力

### 黄金分割支撑阻力

**常用比例：**
- 0.382：弱势回调位
- 0.5：中位数，心理关口
- 0.618：强势回调位，最重要
- 1.618：扩展位，目标位

**计算方法：**
- 上涨后回调：高点 - (高点-低点)×比例
- 下跌后反弹：低点 + (高点-低点)×比例

## 三、实战案例分析

### 案例一：小米集团（1810.HK）整数关口支撑

**时间背景：** 2024年-2025年

**走势特征：**
- 股价在30港元形成强支撑
- 多次测试30元未跌破

**关键价位：**
| 日期 | 最低价 | 收盘价 | 测试次数 | 结果 |
|------|--------|--------|----------|------|
| 2024年8月 | 29.8 | 30.5 | 第1次 | 反弹至35元 |
| 2024年10月 | 29.9 | 31.2 | 第2次 | 反弹至38元 |
| 2024年12月 | 29.7 | 30.8 | 第3次 | 反弹至42元 |
| 2025年2月 | 28.5 | 29.0 | 第4次 | 跌破，趋势转空 |

**交易决策：**
- 前3次测试：30元附近是买入机会
- 第4次跌破：支撑变阻力，30元成为压力位
- 策略转换：从逢低买入转为逢高空

**心理分析：**
- 30港元是整数关口，具有心理支撑
- 多次测试后，市场共识强化
- 最终被跌破，说明空方力量压倒多方

### 案例二：腾讯控股（0700.HK）历史高点阻力

**时间背景：** 2024年

**走势特征：**
- 历史高点475港元形成强阻力
- 多次尝试突破未果

**阻力测试：**
- 3月：涨至474元，遇阻回落
- 5月：涨至476元（假突破），迅速回落
- 7月：涨至473元，三顶形态，大跌

**交易决策：**
- 前两次：靠近475元减仓
- 第三次：形成三顶，清仓
- 后续：股价跌至420元

**关键经验：**
- 历史高点是强阻力
- 假突破后回落更危险
- 多次测试不破，可能大跌

---

### 案例三：贵州茅台（600519.SH）趋势线支撑

**时间背景：** 2024年

**上升趋势线：**
- 连接1月低点1250元和5月低点1380元
- 形成上升趋势线

**走势：**
- 8月股价回调至趋势线附近（约1450元）
- 获得支撑后反弹至1700元
- 11月跌破趋势线（约1550元）
- 随后大跌至1400元

**交易应用：**
- 回踩趋势线：买入机会
- 跌破趋势线：止损离场
- 趋势线支撑有效性：约70%

### 案例四：黄金分割回调位应用

**股票：** 某科技股

**上涨波段：** 从50元涨至100元

**回调支撑位计算：**
- 0.382位：100 - (100-50)×0.382 = 80.9元
- 0.5位：100 - (100-50)×0.5 = 75元
- 0.618位：100 - (100-50)×0.618 = 69.1元

**实际走势：**
- 股价从100元回调至76元（接近0.5位）
- 获得支撑后反弹至90元
- 再次回调至70元（接近0.618位）
- 强势反弹至110元

**策略：**
- 0.382位：弱势回调，谨慎买入
- 0.5位：正常回调，可建仓
- 0.618位：强势回调，重仓买入

## 四、注意事项

### 1. 支撑阻力的强度判断

| 因素 | 强支撑/阻力 | 弱支撑/阻力 |
|------|-------------|-------------|
| 测试次数 | 多次（3次以上） | 1-2次 |
| 时间跨度 | 长期有效 | 短期有效 |
| 成交量 | 大成交量确认 | 小成交量 |
| 整数关口 | 是 | 否 |
| 历史高低点 | 是 | 否 |

### 2. 支撑阻力被突破的判断

**真突破：**
- 收盘价突破（非盘中刺破）
- 突破时放量
- 突破后3日不回落
- 突破幅度>3%

**假突破：**
- 盘中突破但收盘回落
- 突破时缩量
- 迅速回到原区间
- 长上影线/下影线

### 3. 角色互换原则

- **支撑变阻力：** 支撑位被跌破后，反弹时该位成为卖出点
- **阻力变支撑：** 阻力位突破后，回调时该位成为买入点
- **互换确认：** 需要至少一次回踩/反弹确认

## 五、常见误区

### ❌ 误区一：把每个高点/低点都当作支撑阻力
**真相：** 只有多次测试的价位才是有效支撑阻力
- 单次高点/低点意义不大
- 需要至少2-3次测试确认
- **正确做法：** 只关注关键位置

### ❌ 误区二：支撑阻力是精确价位
**真相：** 支撑阻力是一个区域，不是精确点位
- 支撑区：支撑位±2-3%
- 阻力区：阻力位±2-3%
- **正确做法：** 在支撑区分批买入，在阻力区分批卖出

### ❌ 误区三：支撑永远不会破，阻力永远不会突破
**真相：** 支撑阻力终将被突破
- 测试次数越多，被突破概率越大
- 第四次测试后突破概率极高
- **正确做法：** 支撑是买入区，不是铁底；止损必须设

### ❌ 误区四：忽视时间框架
**真相：** 不同时间框架的支撑阻力不同
- 日线支撑 > 小时线支撑
- 周线阻力 > 日线阻力
- **正确做法：** 多时间框架共振最可靠

### ❌ 误区五：单独使用支撑阻力
**真相：** 支撑阻力需要结合其他信号
- 支撑位+KDJ超卖金叉 = 高胜率买入
- 阻力位+放量滞涨 = 高胜率卖出
- **正确做法：** 多指标共振

### ❌ 误区六：突破后立即追涨杀跌
**真相：** 突破后通常有回踩确认
- 突破阻力位：等待回踩确认再买入
- 跌破支撑位：等待反弹确认再卖出
- **正确做法：** 突破不追，回踩确认再进

## 六、进阶技巧

### 1. 多时间框架支撑阻力

**策略：**
- 日线找到主要支撑阻力位
- 小时线找到精细入场点
- 周线判断大趋势方向

**共振效应：**
- 日线支撑 = 小时线支撑 = 买入信号极强
- 日线阻力 = 周线阻力 = 卖出信号极强

### 2. 支撑阻力与成交量

**支撑有效性确认：**
- 回踩支撑时缩量：筹码稳定，支撑有效
- 回踩支撑时放量：恐慌抛售，可能跌破

**阻力突破确认：**
- 突破阻力时放量：真突破概率高
- 突破阻力时缩量：假突破概率高

### 3. 心理关口战法

**常见心理关口：**
- 整数位：10、20、50、100、1000
- 历史高低点
- 发行价、定增价
- 前复权价格

**策略：**
- 跌破整数关口：往往继续下跌
- 突破整数关口：往往继续上涨
- 整数关口附近：多空争夺激烈

### 4. 支撑阻力突破后的目标测算

**等幅测量法：**
- 突破阻力后目标 = 阻力位 + (阻力位 - 前支撑位)
- 跌破支撑后目标 = 支撑位 - (前阻力位 - 支撑位)

**黄金分割扩展：**
- 1.618倍：第一目标位
- 2.618倍：第二目标位

### 5. 假突破识别与利用

**识别假突破：**
- 突破时成交量不足
- 突破后迅速回落
- 长上影线/下影线

**利用假突破：**
- 假突破阻力位：做空机会
- 假跌破支撑位：做多机会
