## 学习方法总结

### 每份技能文档结构
1. **概念定义** - 理解核心思想
2. **计算公式** - 掌握计算原理（文字描述）
3. **实战案例** - 结合实际股票分析
4. **注意事项** - 关键使用要点
5. **常见误区** - 避免典型错误

### 学习节奏
- 每5项技能批量汇报一次
- 持续学习，不停止
- 以理解概念为主，不生成代码
