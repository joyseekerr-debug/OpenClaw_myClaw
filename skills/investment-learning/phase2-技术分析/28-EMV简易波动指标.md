# EMV简易波动指标（Ease of Movement Value）

## 一、概念定义

EMV由Richard Arms Jr.发明，结合价格和成交量，衡量价格移动的"容易程度"。

**核心思想：**
- 价格移动越容易（EMV高），趋势越强
- 价格移动越困难（EMV低），趋势越弱
- 结合量价关系判断趋势质量

**基本原理：**
- 成交量小但价格变动大 = EMV高（容易移动）
- 成交量大但价格变动小 = EMV低（难以移动）

---

## 二、计算公式（文字描述）

### 计算步骤

**第一步：计算中点变动（MID）**
- 当日中点 = (最高价 + 最低价) / 2
- 前日中点 = (前日最高价 + 前日最低价) / 2
- MID = 当日中点 - 前日中点

**第二步：计算箱形比率（BOX RATIO）**
- BOX RATIO = (成交量 / 1亿) / (最高价 - 最低价)
- 注意：成交量通常除以1亿进行标准化

**第三步：计算EMV**
- EMV = MID / BOX RATIO

**第四步：计算EMV的N日移动平均**
- 通常取14日平均

## 三、实战应用

### 应用一：EMV零轴系统
- EMV > 0：多头占优，价格容易上涨
- EMV < 0：空头占优，价格容易下跌
- EMV穿越零轴：趋势转变信号

### 应用二：EMV与价格背离
**顶背离：**
- 价格创新高
- EMV未创新高
- 上涨乏力

**底背离：**
- 价格创新低
- EMV未创新低
- 下跌乏力

### 应用三：EMV趋势确认
- 价格上涨 + EMV上升 = 健康上涨
- 价格上涨 + EMV下降 = 上涨乏力
- 价格下跌 + EMV下降 = 健康下跌
- 价格下跌 + EMV上升 = 下跌乏力

---

## 四、注意事项

### 1. EMV特点
- 无量涨停时EMV极高
- 放量滞涨时EMV极低
- 适合发现量价异常

### 2. 使用建议
- 关注零轴穿越
- 重视背离信号
- 结合价格趋势使用

---

## 五、总结

| 应用场景 | EMV优势 | 注意事项 |
|----------|---------|----------|
| 量价关系分析 | ⭐⭐⭐⭐⭐ | 独特的量价结合 |
| 趋势质量判断 | ⭐⭐⭐⭐ | 容易vs困难 |
| 背离识别 | ⭐⭐⭐⭐ | 信号可靠 |

**一句话总结：** EMV是独特的量价结合指标，衡量价格移动的容易程度，零轴分界多空，背离信号可靠，无量大涨时EMV极高，放量滞涨时EMV极低。
