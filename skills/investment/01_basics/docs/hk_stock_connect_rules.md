# 技能文档：港股通规则详解

## 学习日期
2026-02-24

## 技能概述
港股通是内地投资者交易港股的主要渠道，深入理解其规则对于港股投资至关重要。

---

## 一、港股通基本概念

### 1.1 什么是港股通
港股通是上海证券交易所和深圳证券交易所与香港联合交易所建立的证券交易互联互通机制，允许内地投资者通过A股账户直接买卖港股。

**两大通道：**
- **沪港通**：2014年11月17日开通
- **深港通**：2016年12月5日开通

### 1.2 标的股票范围

**可投资股票（恒生综合大型股、中型股指数成分股）：**
- 恒生综合大型股指数成分股
- 恒生综合中型股指数成分股
- 市值50亿港元以上的恒生综合小型股指数成分股
- A+H股上市公司H股

**不可投资：**
- 深交所上市A股对应的H股（仅限沪股通）
- 联交所上市非H股的其他股票
- 被实施风险警示的股票
- 被暂停上市的股票

## 二、投资者门槛与开户

### 2.1 开户条件（个人投资者）

| 要求 | 具体标准 |
|------|----------|
| **资产门槛** | 证券账户和资金账户内的资产合计不低于50万元人民币（20个交易日日均） |
| **知识测试** | 通过证券公司组织的港股通知识测试（通常80分以上） |
| **风险评估** | 风险承受能力评估结果为稳健型及以上 |
| **诚信记录** | 无严重不良诚信记录 |

### 2.2 开户流程

```
步骤1：查询资格 → 确认20日日均资产≥50万
步骤2：签署协议 → 《港股通委托协议》《风险揭示书》
步骤3：知识测试 → 通过证券公司测试
步骤4：风险测评 → 完成风险承受能力评估
步骤5：开通权限 → T日申请，T+1日可交易
```

## 三、交易规则详解

### 3.1 交易时间（北京时间）

| 时段 | 时间 | 说明 |
|------|------|------|
| **开市前时段** | 9:00-9:30 | 集合竞价，输入竞价盘 |
| **持续交易时段** | 9:30-12:00 | 早市连续竞价 |
| **午间休市** | 12:00-13:00 | 暂停交易 |
| **持续交易时段** | 13:00-16:00 | 午市连续竞价 |
| **收市竞价时段** | 16:00-16:10 | 收盘集合竞价（半日市除外） |

**与A股对比：**
- A股：9:30-11:30, 13:00-15:00
- 港股多1小时（9:00开市，16:10收市）

### 3.2 交易制度

| 规则 | 港股 | A股 |
|------|------|-----|
| **T+0** | ✅ 当日可买卖 | ❌ T+1 |
| **涨跌停** | ❌ 无限制 | ✅ ±10%（ST±5%） |
| **最小单位** | 每手股数由上市公司定（100-2000不等） | 100股/手 |
| **零股交易** | ❌ 一般不允许 | ✅ 允许 |
| **交收制度** | T+2 | T+1 |

### 3.3 订单类型

**竞价盘（开市前/收市时段）：**
- **竞价限价盘**：指定价格，在最终参考平衡价范围内成交
- **竞价市价盘**：按最终参考平衡价成交

**持续交易时段：**
- **限价盘**：指定价格或更优价格成交
- **增强限价盘**：最多同时与10条轮候队伍进行对盘
- **特别限价盘**：没有价格限制，必须全数执行

## 四、费用结构详解

### 4.1 交易费用一览

| 费用名称 | 费率 | 收取方 | 说明 |
|----------|------|--------|------|
| **印花税** | 0.1% | 香港政府 | 买卖双方收取，向上取整到元 |
| **交易费** | 0.005% | 港交所 | 买卖双方 |
| **交易征费** | 0.0027% | 香港证监会 | 买卖双方 |
| **会财局征费** | 0.00015% | 会财局 | 买卖双方 |
| **股份交收费** | 0.002%（最低2港币，最高100港币） | 香港结算 | 买卖双方 |
| **券商佣金** | 0.03%-0.25% | 券商 | 各券商不同，最低通常50港币 |

**费率合计：约0.25%-0.5%（单边）**

### 4.2 费用计算实例

**场景：买入10,000股小米集团，股价35.74 HKD**

```
成交金额 = 10,000 × 35.74 = 357,400 HKD

费用计算：
1. 印花税：357,400 × 0.1% = 357.4 → 358 HKD
2. 交易费：357,400 × 0.005% = 17.87 HKD
3. 交易征费：357,400 × 0.0027% = 9.65 HKD
4. 会财局征费：357,400 × 0.00015% = 0.54 HKD
5. 交收费：357,400 × 0.002% = 7.15 HKD（≥2，≤100）
6. 券商佣金：357,400 × 0.03% = 107.22 HKD（假设）

费用合计 ≈ 500 HKD（约0.14%）
```

## 五、汇率结算机制

### 5.1 双汇率制度

港股通采用**参考汇率**和**结算汇率**双轨制：

| 汇率类型 | 用途 | 特点 |
|----------|------|------|
| **参考汇率** | 交易日冻结资金 | 前一交易日离岸市场中间价±3%（保守估计） |
| **结算汇率** | 实际交收资金 | 交易日收盘后确定，更准确 |

### 5.2 汇率差异处理

**T日交易时：**
- 使用参考汇率冻结资金
- 参考汇率 = 前一交易日离岸市场汇率 × (1±3%)

**T+2交收时：**
- 使用结算汇率实际扣款
- 多退少补

### 5.3 汇率计算实例

**场景：买入10,000股 @ 35.74 HKD**

```
T日冻结（参考汇率0.92）：
冻结金额 = 357,400 HKD × 0.92 = 328,808 CNY

T+2交收（结算汇率0.918）：
实际扣款 = 357,400 HKD × 0.918 = 328,093 CNY

汇率返还 = 328,808 - 328,093 = 715 CNY
```

**如果汇率上涨到0.925：**
```
实际扣款 = 357,400 × 0.925 = 330,595 CNY
差额补扣 = 330,595 - 328,808 = 1,787 CNY
```

## 六、特殊风险提示

### 6.1 汇率风险
- 港币与人民币汇率波动
- 持有期间汇率变动影响实际收益

### 6.2 无涨跌停风险
- 单日涨跌幅可能极大
- 需做好仓位控制

### 6.3 流动性风险
- 港股通标的以外股票无法交易
- 小盘股可能流动性不足

### 6.4 节日差异风险
- 内地与香港节假日不同
- 可能出现一侧开市、一侧休市

### 6.5 公司行为差异
- 投票权行使方式不同
- 分红派息时间差异

## 七、实战操作指南

### 7.1 买入操作流程

```
1. 登录A股账户 → 选择"港股通"
2. 输入股票代码（如：1810.HK）
3. 输入委托价格和数量
4. 确认参考汇率和预计金额
5. 提交订单
6. T+2日查看结算结果
```

### 7.2 查询持仓成本

**真实成本 = 成交价格 + 交易费用 + 汇率成本**

**示例：**
```
买入价：35.74 HKD
交易费用率：0.15%
汇率：0.918

真实成本 = 35.74 × (1 + 0.15%) × 0.918
         = 35.74 × 1.0015 × 0.918
         = 32.84 CNY/股
```

## 八、常见问题

**Q1：港股通和直接开港股账户有什么区别？**
- 港股通：用A股账户，人民币交易，50万门槛
- 港股账户：用港币，需换汇，无门槛（但需香港银行卡）

**Q2：为什么有时候买不了某只港股？**
- 可能不在港股通标的范围内
- 可能被调出标的名单
- 可能处于停牌状态

**Q3：分红如何收取？**
- 港股分红以港币派发
- 自动兑换成人民币存入账户
- 需缴纳10%红利税（港股通特别规定）

## 九、关键要点总结

1. **门槛**：50万资产，知识测试，风险评估
2. **时间**：9:00-16:10，比A股多1小时
3. **制度**：T+0交易，无涨跌停，T+2交收
4. **成本**：总费用约0.25-0.5%
5. **汇率**：参考汇率冻结，结算汇率交收
6. **风险**：汇率波动、无涨跌停、流动性
