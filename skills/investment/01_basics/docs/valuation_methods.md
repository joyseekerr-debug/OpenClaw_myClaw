# 技能文档：估值方法（PE/PB/PS/DCF）

## 学习日期
2026-02-24

## 技能概述
估值是投资决策的核心，掌握多种估值方法可以帮助投资者判断股票的合理价格区间，避免买贵或卖早。

---

## 一、估值基础

### 1.1 为什么要估值

**估值的核心问题：** 这只股票值多少钱？

**估值的目的：**
- 判断当前股价是否合理
- 寻找被低估的投资机会
- 设定合理的买卖价位
- 比较不同投资机会

### 1.2 估值方法分类

| 类型 | 方法 | 适用场景 |
|------|------|----------|
| **相对估值** | PE、PB、PS、PEG | 同行业比较 |
| **绝对估值** | DCF、DDM | 内在价值计算 |
| **资产估值** | NAV、清算价值 | 重资产行业 |

---

## 二、PE估值法（市盈率）

### 2.1 PE基础

**公式：**
```
PE = 股价 / 每股收益(EPS)
或
PE = 总市值 / 净利润
```

**意义：** 投资者愿意为每1元净利润支付多少钱

### 2.2 PE类型

| PE类型 | 计算方法 | 特点 |
|--------|----------|------|
| **静态PE** | 股价 / 上年度EPS | 滞后，不常用 |
| **动态PE（TTM）** | 股价 / 最近4季度EPS | 常用，反映最新业绩 |
| **前瞻PE** | 股价 / 预测EPS | 面向未来，有预测误差 |

### 2.3 PE分析要点

**绝对水平判断：**
| PE范围 | 含义 | 适用行业 |
|--------|------|----------|
| < 10 | 可能被低估 | 银行、地产、传统制造业 |
| 10-20 | 合理区间 | 多数成熟行业 |
| 20-30 | 成长溢价 | 消费、医药、科技 |
| > 30 | 高成长预期 | 高增长行业龙头 |
| > 50 | 泡沫风险 | 需高度警惕 |

**相对比较：**
- 与历史PE比较（当前PE处于历史什么分位）
- 与行业平均比较
- 与可比公司比较

### 2.4 PEG估值

**公式：**
```
PEG = PE / 盈利增长率
```

**判断标准：**
- PEG < 1：可能被低估
- PEG = 1：合理估值
- PEG > 1：可能被高估

**适用：** 成长股估值，弥补PE未考虑增长的缺陷

---

## 三、PB估值法（市净率）

### 3.1 PB基础

**公式：**
```
PB = 股价 / 每股净资产
或
PB = 总市值 / 净资产
```

**意义：** 投资者愿意为每1元净资产支付多少钱

### 3.2 PB适用场景

| 适用行业 | 原因 |
|----------|------|
| **银行** | 资产以市价计，净资产可靠 |
| **保险** | 持有大量金融资产 |
| **地产** | 重资产，土地有价值 |
| **周期性行业** | 盈利波动大，净资产更稳定 |

### 3.3 PB分析要点

**判断标准：**
| PB范围 | 含义 |
|--------|------|
| < 1 | 破净，可能被低估或资产质量问题 |
| 1-2 | 合理区间 |
| 2-3 | 轻资产或高ROE企业 |
| > 3 | 高溢价，需高ROE支撑 |

**ROE-PB关系：**
```
合理PB ≈ ROE / 必要回报率
```

高ROE企业可以支撑高PB，低ROE企业PB应较低。

---

## 四、PS估值法（市销率）

### 4.1 PS基础

**公式：**
```
PS = 总市值 / 营业收入
或
PS = 股价 / 每股营收
```

**意义：** 投资者愿意为每1元营收支付多少钱

### 4.2 PS适用场景

| 适用场景 | 原因 |
|----------|------|
| **亏损企业** | 无法用PE估值 |
| **初创企业** | 利润波动大，营收更稳定 |
| **互联网公司** | 增长优先于盈利 |
| **周期性行业低谷** | 利润为负，用营收估值 |

### 4.3 PS分析要点

**判断标准：**
| PS范围 | 含义 |
|--------|------|
| < 1 | 传统制造业 |
| 1-5 | 一般消费品 |
| 5-10 | 成长型消费/医药 |
| 10-20 | SaaS/高成长互联网 |
| > 20 | 极高增长预期 |

**PS与净利润率关系：**
```
PS = PE × 净利润率
```

高净利润率企业可以支撑高PS。

## 五、DCF估值法（现金流折现）

### 5.1 DCF原理

**核心思想：** 企业价值等于未来自由现金流的现值之和

**公式：**
```
企业价值 = Σ [FCF_t / (1+r)^t] + 终值 / (1+r)^n
```

**关键参数：**
| 参数 | 含义 | 确定方法 |
|------|------|----------|
| **FCF** | 自由现金流 | 经营现金流 - 资本支出 |
| **r** | 折现率（WACC） | 加权平均资本成本 |
| **g** | 永续增长率 | 长期GDP增速 |
| **n** | 预测期 | 通常5-10年 |

### 5.2 DCF计算步骤

**Step 1：预测未来自由现金流**
```
预测期（5-10年）：
- 预测营收增长
- 预测利润率
- 计算自由现金流
```

**Step 2：计算终值**
```
终值 = FCF_n × (1+g) / (r-g)
（戈登增长模型）
```

**Step 3：确定折现率（WACC）**
```
WACC = E/V × Re + D/V × Rd × (1-T)

E: 股权价值
D: 债权价值
V: 总价值(E+D)
Re: 股权成本（CAPM）
Rd: 债务成本
T: 税率
```

**Step 4：计算现值**
```
企业价值 = PV(预测期现金流) + PV(终值)
股权价值 = 企业价值 - 净债务
每股价值 = 股权价值 / 总股本
```

### 5.3 DCF敏感性分析

**关键变量影响：**
| 变量变化 | 估值影响 |
|----------|----------|
| 折现率↑ | 估值↓ |
| 增长率↑ | 估值↑ |
| 永续增长率↑ | 估值↑ |
| 预测期延长 | 估值↑ |

**敏感性分析表示例：**
```
        折现率
        8%    9%    10%   11%
增长 3%  120   105    92    82
率   4%  135   118   103    91
     5%  155   135   118   103
```

## 六、估值方法选择

### 6.1 按行业选择

| 行业 | 首选估值法 | 次选估值法 |
|------|------------|------------|
| **银行/保险** | PB | PE |
| **地产** | NAV | PB |
| **消费** | PE | PEG |
| **医药** | PE/PEG | PS |
| **科技/互联网** | PS/PE | PEG |
| **周期性行业** | PB | PS |
| **亏损企业** | PS | PB |
| **稳定增长** | DCF | PE |

### 6.2 按企业阶段选择

| 阶段 | 特征 | 推荐估值法 |
|------|------|------------|
| **初创期** | 无盈利，高增长 | PS |
| **成长期** | 盈利增长快 | PEG、前瞻PE |
| **成熟期** | 盈利稳定 | PE、DCF |
| **衰退期** | 盈利下滑 | PB、资产估值 |

## 七、实战案例

### 案例1：消费股PE估值

**背景：** 贵州茅台估值分析

**数据：**
- 股价：1600元
- TTM每股收益：58元
- 当前PE = 1600 / 58 = 27.6倍

**分析：**
```
历史PE区间：20-40倍
当前位置：历史中位偏上
行业平均PE：25倍
结论：估值合理偏高，未显著低估
```

### 案例2：银行股PB估值

**背景：** 招商银行估值分析

**数据：**
- 股价：35元
- 每股净资产：32元
- 当前PB = 35 / 32 = 1.09倍
- ROE = 15%

**分析：**
```
行业平均PB：0.8倍
招行历史PB：0.9-1.5倍
合理PB估算：ROE/必要回报率 = 15%/10% = 1.5倍
结论：当前PB低于合理值，有一定安全边际
```

### 案例3：科技股PEG估值

**背景：** 宁德时代估值分析

**数据：**
- 股价：200元
- TTM EPS：6元
- 当前PE = 33.3倍
- 预期增长率：30%

**计算：**
```
PEG = 33.3 / 30 = 1.11
```

**分析：**
```
PEG > 1，估值偏高
但考虑到：
- 行业龙头地位
- 新能源车渗透率提升
- 储能业务增长
结论：估值合理，需持续跟踪业绩增长
```

### 案例4：互联网公司PS估值

**背景：** 美团估值分析

**数据：**
- 市值：8000亿港币
- 营业收入：2700亿人民币
- 当前PS = 8000/2700 = 3.0倍
- 收入增长率：25%

**分析：**
```
可比公司PS：2-5倍
当前位置：行业中位
考虑增长：PS/G = 3/25 = 0.12
结论：估值相对合理
```

### 案例5：DCF估值实例

**背景：** 腾讯控股DCF估值

**假设：**
```
当前自由现金流：1500亿
预测期：10年
前5年增长率：15%
后5年增长率：10%
永续增长率：3%
WACC：10%
净现金：2000亿
总股本：95亿股
```

**计算结果：**
```
预测期现金流现值：8500亿
终值现值：12000亿
企业价值：20500亿
股权价值：22500亿
每股价值：237元
当前股价：380港币（约350人民币）
```

**结论：**
```
DCF估值237元 < 当前股价350元
可能原因：
- 市场给予更高增长预期
- 或当前股价已透支未来增长
- DCF参数偏保守
```

## 八、估值陷阱

### 陷阱1：低PE陷阱

**表现：** PE很低，但盈利即将大幅下滑

**案例：** 周期性行业在周期高点PE很低，但即将进入下行周期

### 陷阱2：高成长陷阱

**表现：** 给予过高增长预期，实际无法兑现

**案例：** 某些概念股炒作，预期100%增长，实际仅10%

### 陷阱3：DCF参数游戏

**表现：** 通过调整WACC、增长率来得到想要的估值

**警示：** DCF结果对参数高度敏感，需谨慎设定

### 陷阱4：忽视质量

**表现：** 只看估值数字，忽视企业质地

**警示：** 好公司贵有贵的道理，烂公司便宜也可能陷阱

## 九、估值实践建议

### 9.1 多方法交叉验证

**建议：** 至少使用2-3种估值方法，相互验证

**示例：**
- 消费股：PE + PEG + DCF
- 银行股：PB + PE
- 科技股：PS + PEG

### 9.2 安全边际原则

**格雷厄姆原则：**
```
买入价格 ≤ 内在价值 × 0.7
（30%安全边际）
```

### 9.3 动态调整

**估值不是一成不变：**
- 定期更新财务数据
- 跟踪业绩变化
- 调整增长预期

## 学习总结

**核心要点：**
1. PE：最常用，盈利稳定企业适用
2. PB：周期股、金融机构适用
3. PS：亏损企业、高成长企业适用
4. DCF：绝对估值，参数敏感
5. PEG：成长股估值，弥补PE不足
6. 多方法交叉验证，避免单一估值偏见
7. 留足安全边际，不追求精确到小数点
