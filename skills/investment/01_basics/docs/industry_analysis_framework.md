## 学习日期
2026-02-24

## 技能概述
行业分析是投资决策的重要环节，了解行业的竞争格局、发展阶段和关键成功因素，有助于筛选优质赛道和识别投资机会。

---

## 一、行业分析基础

### 1.1 为什么要做行业分析

**行业研究的目的：**
- 识别优质赛道（好行业）
- 避开衰退行业
- 理解竞争格局
- 把握投资时机

**好行业的特征：**
- 市场规模大且持续增长
- 竞争格局清晰，龙头集中
- 进入壁垒高
- 盈利能力强
- 政策支持或不受政策打压

### 1.2 行业分类体系

**按生命周期分类：**
| 阶段 | 特征 | 代表行业 | 投资策略 |
|------|------|----------|----------|
| **导入期** | 技术不成熟，市场小 | 人工智能、量子计算 | 高风险高回报 |
| **成长期** | 快速增长，渗透率提升 | 新能源汽车、光伏 | 投资黄金期 |
| **成熟期** | 增速放缓，格局稳定 | 白酒、家电、银行 | 关注龙头、分红 |
| **衰退期** | 需求萎缩，产能过剩 | 传统煤炭、纸媒 | 回避或博弈反弹 |

**按周期性分类：**
| 类型 | 特征 | 代表行业 | 投资要点 |
|------|------|----------|----------|
| **周期型** | 盈利随经济周期波动 | 钢铁、有色、化工、航运 | 逆周期布局 |
| **防御型** | 需求稳定，抗周期 | 公用事业、食品饮料、医药 | 避险配置 |
| **成长型** | 独立于周期，持续增长 | 科技、互联网、生物医药 | 长期持有 |

## 二、行业分析框架（波特五力+）

### 2.1 波特五力模型

```
        新进入者威胁
              ↓
供应商议价能力 → [行业内竞争] → 买方议价能力
              ↑
        替代品威胁
```

### 2.2 五力详细分析

#### 1) 行业内竞争

**竞争激烈程度判断：**
| 因素 | 竞争激烈 | 竞争温和 |
|------|----------|----------|
| 企业数量 | 多 | 少 |
| 行业增速 | 慢 | 快 |
| 产品差异化 | 低 | 高 |
| 退出壁垒 | 高 | 低 |
| 固定成本占比 | 高 | 低 |

**案例：**
- 竞争激烈：快递行业（价格战持续）
- 竞争温和：高端白酒（品牌壁垒高）

#### 2) 新进入者威胁

**进入壁垒类型：**
| 壁垒类型 | 说明 | 例子 |
|----------|------|------|
| **规模经济** | 大企业成本优势 | 汽车制造 |
| **品牌壁垒** | 消费者忠诚度 | 可口可乐 |
| **渠道壁垒** | 分销网络控制 | 农夫山泉 |
| **技术壁垒** | 专利、know-how | 芯片制造 |
| **资金壁垒** | 巨额资本投入 | 航空、铁路 |
| **政策壁垒** | 牌照、准入许可 | 银行、保险 |

#### 3) 替代品威胁

**威胁程度判断：**
| 高威胁 | 低威胁 |
|--------|--------|
| 替代品性价比更高 | 产品独特性强 |
| 转换成本低 | 转换成本高 |
| 技术迭代快 | 技术稳定 |

**案例：**
- 高威胁：传统相机（被手机替代）
- 低威胁：茅台（无真正替代品）

#### 4) 供应商议价能力

**供应商强势信号：**
- 供应商集中度高
- 产品差异化强
- 转换成本高
- 前向整合威胁

**案例：**
- 强势供应商：芯片制造商（对下游手机厂）
- 弱势供应商：普通原料供应商

#### 5) 买方议价能力

**买方强势信号：**
- 买方集中度高（大客户）
- 产品标准化
- 转换成本低
- 价格敏感度高

**案例：**
- 强势买方：沃尔玛（对供应商）
- 弱势买方：散客消费者

### 2.3 第六力：政策/监管力量

**在中国市场的特殊重要性：**
| 政策类型 | 影响 | 例子 |
|----------|------|------|
| **产业支持** | 利好 | 新能源补贴 |
| **产业限制** | 利空 | 房地产调控 |
| **价格管制** | 压缩利润 | 药品集采 |
| **准入监管** | 影响供给 | 教培行业整顿 |

## 三、行业关键指标

### 3.1 市场规模与增速

| 指标 | 说明 | 数据来源 |
|------|------|----------|
| **TAM** | 总潜在市场 | 行业报告 |
| **SAM** | 可服务市场 | 公司披露 |
| **SOM** | 可获得市场 | 公司披露 |
| **CAGR** | 复合年增长率 | 计算得出 |

**判断标准：**
- CAGR > 20%：高速增长
- CAGR 10-20%：稳健增长
- CAGR 5-10%：低速增长
- CAGR < 5%：接近成熟

### 3.2 盈利能力指标

| 指标 | 计算方法 | 行业对比 |
|------|----------|----------|
| **行业平均毛利率** | 样本企业平均 | 越高越好 |
| **行业平均ROE** | 样本企业平均 | >15%优秀 |
| **行业平均费用率** | 销售/管理/研发费用 | 越低越好 |

### 3.3 集中度指标

**CRn（行业集中度）：**
```
CR4 = 前4大企业市场份额之和
CR8 = 前8大企业市场份额之和
```

**判断标准：**
| CR4 | 竞争格局 | 投资策略 |
|-----|----------|----------|
| > 60% | 寡头垄断 | 投资龙头 |
| 40-60% | 垄断竞争 | 关注差异化 |
| < 40% | 完全竞争 | 难以投资 |

## 四、重点行业分析

### 4.1 消费行业

**细分赛道：**
| 细分 | 特征 | 代表公司 |
|------|------|----------|
| **白酒** | 品牌壁垒高，盈利极强 | 茅台、五粮液 |
| **调味品** | 刚需，复购率高 | 海天味业 |
| **乳制品** | 双寡头格局 | 伊利、蒙牛 |
| **休闲食品** | 竞争激烈，同质化 | 三只松鼠 |

**关键成功因素：**
- 品牌力
- 渠道深度
- 产品创新能力

### 4.2 医药行业

**细分赛道：**
| 细分 | 特征 | 投资要点 |
|------|------|----------|
| **创新药** | 高投入高风险高回报 | 管线、临床进度 |
| **仿制药** | 集采影响大 | 成本控制、集采中标 |
| **医疗器械** | 进口替代空间大 | 技术突破、注册进度 |
| **CXO** | 医药外包服务 | 订单增长、产能扩张 |

**关键成功因素：**
- 研发能力
- 管线丰富度
- 商业化能力

### 4.3 科技行业

**细分赛道：**
| 细分 | 特征 | 代表公司 |
|------|------|----------|
| **半导体** | 国产替代主线 | 中芯国际 |
| **互联网** | 平台效应强 | 腾讯、阿里 |
| **新能源** | 政策+需求双驱动 | 宁德时代 |
| **软件服务** | 订阅模式，现金流好 | 金山办公 |

**关键成功因素：**
- 技术创新
- 用户规模
- 生态构建

### 4.4 金融行业

**细分赛道：**
| 细分 | 特征 | 投资要点 |
|------|------|----------|
| **银行** | 高杠杆，稳定分红 | 资产质量、息差 |
| **保险** | 长期负债经营 | 新业务价值、投资端 |
| **券商** | 周期性明显 | 市占率、业务结构 |
| **资管** | 轻资产，高ROE | AUM增长、费率 |

**关键成功因素：**
- 风险控制
- 客户基础
- 专业能力

## 六、实战案例

### 案例1：白酒行业分析

**行业概况：**
- 市场规模：6000亿+
- CAGR：5-8%
- 竞争格局：一超（茅台）+ 多强

**五力分析：**
```
- 行业内竞争：温和（差异化明显，不打价格战）
- 进入壁垒：极高（品牌需时间积累）
- 替代品威胁：低（社交属性难替代）
- 供应商议价力：低（粮食等大宗商品）
- 买方议价力：低（品牌忠诚度高）
```

**关键指标：**
- 茅台毛利率：91%
- 行业平均ROE：20%+
- CR3：约40%

**结论：** 优质赛道，适合长期配置

### 案例2：新能源汽车行业分析

**行业概况：**
- 市场规模：万亿级
- CAGR：30%+
- 发展阶段：成长期

**五力分析：**
```
- 行业内竞争：激烈（价格战持续）
- 进入壁垒：中等（资金+技术）
- 替代品威胁：低（趋势不可逆）
- 供应商议价力：分化（电池强势）
- 买方议价力：高（选择多）
```

**关键指标：**
- 行业平均毛利率：15-20%
- 渗透率：30%（快速提升中）
- CR5：约60%

**结论：** 高速增长但竞争激烈，优选龙头

### 案例3：教培行业政策影响

**政策变化：** 2021年"双减"政策

**影响分析：**
```
- 学科类培训：基本归零
- 素质教育：相对受益
- 成人教育：影响较小
- 教育信息化：受益
```

**投资启示：** 政策风险需高度重视

## 八、常见误区

### 误区1：只看行业好就买
**纠正：** 好行业也需要好价格，且要选优质公司

### 误区2：忽视行业周期
**纠正：** 成长期买入，成熟期持有，衰退期回避

### 误区3：过度关注短期波动
**纠正：** 关注行业长期趋势和结构性变化

### 误区4：忽视政策风险
**纠正：** 中国市场政策影响巨大，需持续跟踪

## 学习总结

**核心要点：**
1. 行业分析选赛道，公司分析选个股
2. 波特五力评估竞争格局
3. 生命周期判断投资时机
4. 关注市场规模、增速、盈利能力、集中度
5. 政策力量在中国尤为重要
6. 好行业 = 大空间 + 好格局 + 高盈利 + 政策支持
