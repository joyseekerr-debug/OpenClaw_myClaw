## 一、概念定义

### 1.1 什么是股票市场
股票市场是股票发行和交易的场所，是企业融资和投资者投资的平台。

**核心功能：**
- **融资功能**：企业通过发行股票筹集资金
- **投资功能**：投资者通过买卖股票获得收益
- **定价功能**：市场供求关系决定股票价格
- **资源配置**：引导资金流向优质企业

### 1.2 市场参与者

| 参与者 | 角色 | 目的 |
|--------|------|------|
| **上市公司** | 股票发行方 | 融资扩张 |
| **投资者** | 股票买卖方 | 资本增值 |
| **券商** | 交易中介 | 佣金收入 |
| **交易所** | 交易场所 | 提供平台 |
| **监管机构** | 市场监管 | 维护秩序 |

## 二、市场结构

### 2.1 一级市场 vs 二级市场

**一级市场（发行市场）：**
- IPO（首次公开发行）
- 增发、配股
- 可转债发行
- **特点**：发行人直接融资，投资者从公司购买

**二级市场（流通市场）：**
- 股票在投资者之间买卖
- 不影响公司股本
- **特点**：提供流动性，价格由供需决定

### 2.2 交易场所

**交易所市场（场内市场）：**
- 沪深交易所（A股）
- 港交所（港股）
- 纽交所/纳斯达克（美股）
- **特点**：集中竞价，透明度高

**场外市场（OTC）：**
- 新三板
- 区域性股权市场
- **特点**：协议转让，流动性差

## 三、交易机制

### 3.1 交易时间

**A股市场：**
- 开盘集合竞价：9:15-9:25
- 连续竞价：9:30-11:30, 13:00-15:00
- 收盘集合竞价：14:57-15:00

**港股市场：**
- 开市前时段：9:00-9:30
- 持续交易：9:30-12:00, 13:00-16:00
- 收市竞价：16:00-16:10

**关键区别：**
- A股：T+1交易，涨跌停限制（±10%）
- 港股：T+0交易，无涨跌停，T+2交收

### 3.2 价格形成机制

**集合竞价：**
```
成交原则：
1. 最大成交量原则
2. 高于成交价的买入申报全部成交
3. 低于成交价的卖出申报全部成交
```

**连续竞价：**
```
成交原则：
1. 价格优先：高价买、低价卖优先
2. 时间优先：同价时先申报优先
```

### 3.3 订单类型

| 订单类型 | 说明 | 使用场景 |
|----------|------|----------|
| **限价单** | 指定价格成交 | 控制成本 |
| **市价单** | 当前最优价成交 | 急需成交 |
| **止损单** | 触发价后转为市价 | 风险控制 |
| **条件单** | 满足条件后触发 | 自动交易 |

## 四、港股通特殊规则

### 4.1 交易规则
- **交易货币**：港币报价，人民币交收
- **汇率结算**：参考汇率 vs 结算汇率
- **碎股交易**：不足一手的碎股交易受限

### 4.2 税费结构

| 费用 | 费率 | 说明 |
|------|------|------|
| **印花税** | 0.1% | 买卖双方 |
| **交易费** | 0.005% | 港交所收取 |
| **交收费** | 0.002% | 香港结算 |
| **交易征费** | 0.0027% | 证监会 |
| **佣金** | 0.03-0.25% | 券商收取 |

**总计约：0.25-0.5%**

### 4.3 投资者门槛
- 账户资产≥50万人民币（20个交易日）
- 通过港股通知识测试
- 风险承受能力评估

### 5.1 港股主要指数

|------|--------|------|

### 5.2 指数计算
```
恒生指数 = Σ(成分股市值 × 股价) / 除数

权重调整：每季度调整一次
成分股调整：根据市值、流动性筛选
```

## 六、实战案例

### 案例1：理解撮合机制

**场景：**
- 买1：10.00元 1000股
- 买2：9.99元 2000股
- 卖1：10.01元 1500股
- 卖2：10.02元 2000股

**分析：**
- 当前无法成交（买价 < 卖价）
- 若有新买单10.01元500股 → 与卖1成交
- 成交后：买1仍为10.00元，卖1剩余1000股

### 案例2：港股通汇率计算

**场景：**
- 买入：1000股 @ 35.00 HKD
- 参考汇率：1 HKD = 0.9200 CNY
- 结算汇率：1 HKD = 0.9180 CNY

**计算：**
```
冻结资金 = 1000 × 35.00 × 0.9200 = 32,200 CNY
实际支付 = 1000 × 35.00 × 0.9180 = 32,130 CNY
汇率收益 = 32,200 - 32,130 = 70 CNY
```

## 七、常见误区

### 误区1：T+0就是可以随时买卖
**纠正**：T+0指当天可买卖，但资金T+2才能取出

### 误区2：集合竞价价格就是开盘价
**纠正**：只有成交的价格才是开盘价，未成交的委托作废

### 误区3：港股没有涨跌停就没有风险
**纠正**：无涨跌停意味着单日波动可能极大，风险更高
