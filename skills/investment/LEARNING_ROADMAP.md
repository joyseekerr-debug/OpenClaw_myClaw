# 投资技能学习路线图与计划

## 学习目标
系统性掌握炒股及投资完整技能清单（200+技能点），每个技能深入理解并归档到技能库。

## 学习阶段规划

### Phase 1: 基础认知（第1-2周）
- [ ] 股票市场运作机制
- [ ] 港股通规则详解
- [ ] 财务报表阅读
- [ ] 估值方法（PE/PB/PS/DCF）
- [ ] 行业分析框架
- [ ] 宏观经济基础

### Phase 2: 技术分析基础（第3-5周）
- [ ] 趋势分析（均线、趋势线）
- [ ] 动量指标（MACD/RSI/KDJ）
- [ ] 趋势指标（布林带/ATR）
- [ ] 成交量指标（OBV）
- [ ] 形态分析（K线形态）

### Phase 3: 技术分析进阶（第6-8周）
- [ ] 高级形态（头肩/双顶/圆弧）
- [ ] 多时间尺度分析
- [ ] 支撑阻力精确计算
- [ ] 技术指标组合策略

### Phase 4: 量化技能（第9-14周）
- [ ] Python数据处理精进
- [ ] 特征工程（Alpha因子挖掘）
- [ ] 机器学习模型（XGBoost/LSTM）
- [ ] 回测框架搭建
- [ ] 绩效评估体系

### Phase 5: 风险管理（第15-17周）
- [ ] 仓位管理策略
- [ ] 止损止盈体系
- [ ] 组合风险管理
- [ ] 压力测试

### Phase 6: 交易执行（第18-20周）
- [ ] 订单类型与时机
- [ ] 成本控制
- [ ] 滑点管理

### Phase 7: 监控预警（第21-22周）
- [ ] 实时监控系统
- [ ] 预警算法
- [ ] 通知系统优化

### Phase 8: 投资心理（持续）
- [ ] 情绪管理
- [ ] 决策框架
- [ ] 复盘机制

### Phase 9: 进阶技能（第23-30周）
- [ ] 衍生品基础
- [ ] 跨市场套利
- [ ] 另类数据

## 学习产出要求

每个技能学习后必须产出：
1. **技能文档** (`skills/investment/SKILL_NAME.md`)
   - 概念定义
   - 计算公式
   - 实战案例
   - 代码实现
   - 注意事项

2. **实战代码** (`skills/investment/code/`)
   - 可运行的Python代码
   - 单元测试
   - 使用示例

3. **测试验证** (`skills/investment/tests/`)
   - 测试用例
   - 验证结果

4. **学习笔记** (`memory/learning/`)
   - 学习心得
   - 常见误区
   - 进阶技巧

- [ ] 概念理解清晰

- [ ] 有3+实战案例
- [ ] 代码可运行
- [ ] 文档完整
- [ ] 通过测试验证

## 每日学习节奏

- **每日**: 学习1-2个技能
- **每周**: 完成一个技能类别
- **每月**: 完成一个Phase
- **每季度**: 系统性复习
