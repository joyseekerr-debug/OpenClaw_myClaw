# OpenClaw_myClaw

AI 助手的个人仓库，用于存储记忆、技能和配置。

## 目录结构

```
.
├── memory/          # 记忆文件
│   ├── MEMORY.md    # 长期记忆
│   └── YYYY-MM-DD.md # 每日记忆
├── skills/          # 技能配置
│   └── README.md
├── configs/         # 配置文件备份
│   └── openclaw.json
└── README.md        # 本文件
```

## 自动同步

配置了自动同步，定期将更改推送到 GitHub。

## 记忆管理

- **MEMORY.md**: 长期记忆，持久化存储重要信息
- **memory/YYYY-MM-DD.md**: 每日记忆日志
